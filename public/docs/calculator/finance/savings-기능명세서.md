# savings-기능명세서.md

## 1. 개요
적금 이자 계산 메뉴의 기능 명세서입니다. 매월 일정액을 저축하여 만기 시 받을 수 있는 이자와 수령액을 계산합니다.

## 2. 화면 위치
1. 경로 : `/calculator/finance/savings`
2. 동적 경로 (SEO) : `/calculator/finance/savings/[한글슬러그]` (예: /정기적금, /자유적금 등)
3. 파일
ㄴ 서버 : `app/calculator/finance/savings/page.tsx`
ㄴ 동적 서버 : `app/calculator/finance/savings/[slug]/page.tsx`
ㄴ 클라이언트 : `app/calculator/finance/savings/Savings.tsx`
ㄴ 데이터 : `app/calculator/finance/data/products.json` (한글 슬러그 및 상품 정보 관리)

## 3. 계층 구조 및 SEO (Common Architecture)
1. **Breadcrumb Schema**: `홈 > 계산기 홈 > 금융 계산기 > 적금 계산기` 구조의 JSON-LD 적용
2. **동적 메타데이터**: `[slug]`(적금명)에 따라 Title, Description, Keywords 자동 생성
3. **한글 슬러그 처리**: URL 파라미터를 `decodeURIComponent` 및 `normalize('NFC')` 처리하여 검색엔진 유입 최적화.

## 4. 화면 기능
1. **입력 데이터**
   - **월 적립액(원)**: 매달 저축하는 금액 (단위 버튼: 1만, 5만, 10만, 50만, 100만 등 제공)
   - **적립 기간(개월/년)**: 6, 12, 24, 36개월 등 단축 버튼 제공
   - **연 이자율(%)**: 소수점 2자리까지 입력 가능
   - **이자 계산 방식**: 단리, 월복리 선택
   - **과세 유형**: 일반과세(15.4%), 세금우대(9.5%), 비과세(0%)

2. **출력 데이터**
   - **세후 수령액**: 원금과 세후 이자의 합계 (강조 노출)
   - **원금 합계**: (월 적립액 × 기간)
   - **세전 이자**: 이자 과세 전 총액
   - **이자 소득세**: 세율 및 차감액 표시
   - **세후 이자**: 실제 수령하는 이자액

3. **UI/UX 기능**
   - **상단 상품 배지**: `🏦 [상품명]` 형태의 배지 노출
   - **바 차트**: 원금과 이자의 비율을 시각적으로 비교 제공
   - **결과 복사**: 계산 결과를 텍스트 형식으로 클립보드에 복사

## 5. 하단 콘텐츠 (SEO 가이드)
1. **소개**: 목돈 마련을 위한 적금의 목적과 장점 안내
2. **사용 방법**: 입력 항목별 가이드 및 팁 제공
3. **계산 예시**: 100만원씩 1년, 연 5% 적립 시의 구체적인 결과 예시
4. **FAQ**: 예금과 적금의 이자 차이 등 사용자 궁금증 해소 (**공통 `FAQ` 컴포넌트**)
5. **상품 리스트 확대**: 정기, 자유, 자유적립식, 정액적립식, 비과세, 주택청약종합저축 등 지원

## 6. 계산 로직 (단리 기준)
- **월별 이자** = 월 적립액 × 이율/100 × (남은 개월 수 / 12)
- **총 세전 이자** = 모든 월별 이자의 합계
- **세후 이자** = 총 세전 이자 × (1 - 과세율)
- **만기 수령액** = 원금 합계 + 세후 이자

# loans-기능명세서.md

## 1. 개요
대출 이자 계산 메뉴의 기능 명세서입니다. 다양한 상환 방식에 따른 비용 차이를 비교하고 상환 플랜을 설계할 수 있습니다.

## 2. 화면 위치
1. 경로 : `/calculator/finance/loans`
2. 동적 경로 (SEO) : `/calculator/finance/loans/[한글슬러그]` (예: /주택담보대출, /신용대출 등)
3. 파일
ㄴ 서버 : `app/calculator/finance/loans/page.tsx`
ㄴ 동적 서버 : `app/calculator/finance/loans/[slug]/page.tsx`
ㄴ 클라이언트 : `app/calculator/finance/loans/Loans.tsx`
ㄴ 데이터 : `app/calculator/finance/data/products.json` (한글 슬러그 및 상품 정보 관리)

## 3. 계층 구조 및 SEO (Common Architecture)
1. **Breadcrumb Schema**: `홈 > 계산기 홈 > 금융 계산기 > 대출 계산기` 구조의 JSON-LD 적용
2. **동적 메타데이터**: `[slug]`(대출 상품명)에 따라 Title, Description, Keywords 자동 생성
3. **한글 슬러그 처리**: URL 파라미터를 `decodeURIComponent` 및 `normalize('NFC')` 처리하여 검색엔진 유입 최적화.

## 4. 화면 기능
1. **입력 데이터**
   - **대출 금액(원)**: (단위 버튼: 100만, 1000만, 5000만, 1억 등 제공)
   - **대출 기간(개월/년)**: **6, 12, 24, 36개월 단축 선택 버튼**을 통한 사용자 편의성 증대
   - **연 이자율(%)**: 소수점 2자리 입력
   - **상환 방식**: 원리금균등, 원금균등, 만기일시상환 선택
   - **거치 기간(개월)**: 원금 상환 없이 이자만 납부하는 기간 설정

2. **출력 데이터**
   - **총 상환 금액**: 대출 원금 + 총 이자 합계
   - **총 대출 이자**: 대출 기간 동안 납부하는 총 이자액
   - **월 평균 상환액**: 매달 납부해야 하는 평균 금액
   - **회차별 상세 내역**: 월별 원금 상환액, 이자액, 잔액 변화를 표 형태로 제공

3. **UI/UX 기능**
   - **상단 상품 배지**: `💰 [상품명]` 형태의 배지 노출
   - **상환 스케줄 그래프**: 기간 경과에 따른 대출 잔액의 감소 추세를 시각화
   - **결과 복사**: 계산 결과를 클립보드에 복사

## 5. 하단 콘텐츠 (SEO 가이드)
1. **소개**: 대출 상환 방식(원리금균등 vs 원금균등)에 따른 장단점 안내
2. **사용 방법**: 거치 기간 설정 및 상환 방식 선택 가이드
3. **계산 예시**: 1억 대출, 연 4% 이자, 30년 상환 시의 구체적 상환 스케줄 예시
4. **FAQ**: 중도상환수수료 및 상환 방식별 이자 차이 설명 (**공통 `FAQ` 컴포넌트**)
5. **상품 리스트 확대**: 신용, 개인신용, 직장인, 햇살론, 주택담보, 전세자금, 자동차담보, 예금담보, 보험계약, 디딤돌, 버팀목 등 지원

## 6. 계산 로직 (원리금균등 상환 기준)
- **월 상환액** = [대출원금 × 월금리 × (1 + 월금리)^기간] / [(1 + 월금리)^기간 - 1]
- *월금리 = 연이율 / 12 / 100*
- **총 이자** = (월 상환액 × 기간) - 대출원금

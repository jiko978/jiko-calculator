# 퍼센트 계산기 기능 명세서 (Percentage Calculator)

## 1. 개요
* **목적**: 투자 수익률, 주식/부동산 비중, 세금 계산 등 다양한 금융 계산에서 가장 기본이 되는 퍼센트(%) 계산을 빠르고 정확하게 수행합니다.
* **경로**: `/calculator/finance/percentage`
* **UI/UX 원칙**: JIKO 표준 (카드형 레이아웃, `max-w-3xl`, Smooth Scroll, 공통 하단 컴포넌트)

## 2. 화면 위치 및 파일 구조
1. **경로**: `/calculator/finance/percentage`
2. **파일 구조**:
   - 서버 : `app/calculator/finance/percentage/page.tsx`
   - 클라이언트 : `app/calculator/finance/percentage/Percentage.tsx`

## 3. 계층 구조 및 SEO (Common Architecture)
1. **Breadcrumb**: `홈 > 계산기 홈 > 금융 계산기 > 퍼센트 계산기`
2. **JSON-LD**: `SoftwareApplication` 및 `FAQPage` 스키마 적용.

## 4. JIKO만의 차별화 기능 (Differentiation)
1. **4 in 1 실시간 다중 모드 연산**: 
   - 비율 계산, 일부값 계산, 증감값 계산, 증감율 계산 등 사용자가 가장 많이 찾는 4가지 퍼센트 계산 공식을 한 화면에 독립적인 카드로 구성합니다.
   - 탭 이동 없이 원하는 계산기의 입력란에 숫자를 넣으면 즉시 결과가 도출되는 **실시간 연산(On-change 연산)** UX를 적용합니다.
2. **금융/생활 맞춤 예시 제공**: 
   - 단순히 숫자만 계산하는 것을 넘어 "주식 수익률", "쇼핑몰 할인율", "세금(VAT) 계산" 등 사용자가 친숙한 예제를 Placeholder나 작은 안내 문구로 삽입하여 직관성을 극대화합니다.
3. **단위 입력 편의성 향상**:
   - `만`, `억` 단위 버튼 대신, 퍼센트 특성에 맞게 `%` 와 일반 숫자를 혼용하기 쉽도록 천 단위 콤마(`,`) 자동 포맷팅과 `+10%`, `-20%` 등의 빠른 증감 버튼을 제공합니다.

## 5. 화면 기능
### 5.1 입력 데이터 (4가지 독립 모드)
각 모드별로 2개의 입력값을 받습니다.
1) **비율값 계산**: 전체값의 [ A ]%는 얼마인가? (입력: 전체값, 비율값)
2) **일부값 계산**: 전체값에서 [ B ]은(는) 몇 %인가? (입력: 전체값, 일부값)
3) **증감값 계산**: 전체값에서 [ C ]으로 변하면 몇 % 증감했는가? (입력: 전체값, 변경값)
4) **증감율 계산**: 전체값이 [ D ]% 증가/감소하면 얼마인가? (입력: 전체값, 증감비율값)

### 5.2 출력 데이터
각 입력 폼 바로 아래에 직관적인 결과 표시.
1) ➡️ [전체값]의 [ A ]%는 **[결과값]** 입니다.
2) ➡️ [전체값]에서 [ B ]은(는) **[결과값]%** 입니다.
3) ➡️ [전체값]에서 [ C ]으로 변하면 **[결과값]% [증가/감소]** 하였습니다.
4) ➡️ [전체값]에서 [ D ]% [증가/감소]하면 **[결과값]** 입니다.

### 5.3 버튼 및 UX
- **CalculatorButtons**: 표준 초기화/계산하기 버튼.
- **실시간 결과 도출**: 입력 필드 `onChange` 발생 시 즉시 하단에 결과를 렌더링.


## 6. 결과 화면 및 시각화 (UI/UX)
* **CalculatorActions**: 카카오톡 공유 및 결과 클립보드 복사.
* **Red Box 에러 톤앤매너**: 잘못된 수식(0으로 나누기 등) 발생 시 JIKO 표준의 부드러운 경고 메시지 표출.

## 7. 하단 콘텐츠 (SEO 및 가이드)
정보 제공 섹션은 딱딱한 줄글을 지양하고, **JIKO만의 카드(Card) 레이아웃(1.6~1.10 표준 패턴)**을 완벽히 준수하여 섹션별로 분리합니다.

### 1.6 퍼센트 계산기 가이드 (1단 레이아웃)
- **아이콘 및 제목**: 💯 퍼센트 계산기 가이드 (비율, 할인율, 수익률 자동 연산)
- **내용**: 쇼핑할 때 할인율 계산부터, 주식/코인 투자 시 수익률 계산, 부동산 세금 등 일상생활과 금융 전반에서 가장 많이 쓰이는 4가지 퍼센트 공식을 한 번에 제공합니다. 복잡한 수식 없이 숫자만 입력하면 즉시 답을 찾아드립니다.

### 1.7 사용 방법 및 계산 예시 (2단 그리드)
- **💡 사용 방법 (좌측)**: 
  - 4가지 모드 중 내가 원하는 계산 방식을 확인합니다.
  - 빈칸에 숫자(전체값, 비율, 일부값 등)를 입력합니다.
  - 버튼을 누를 필요 없이 입력과 동시에 아래에 결과가 표시됩니다.
- **📊 계산 예시 (우측)**: 
  - (할인율) 50,000원짜리 옷이 20% 할인하면? -> 40,000원!
  - (수익률) 10,000원에 산 주식이 13,000원이 되면? -> 30% 상승!

### 1.8 퍼센트 활용 꿀팁 (2단 그리드 - 추가 카드 세션 2개)
- **카드 1: 📌 생활 속 퍼센트 활용법 (아이콘: 🛒)**
  - 쇼핑몰에서 "30% 할인 + 추가 10% 쿠폰"은 총 40% 할인이 아닙니다! (0.7 × 0.9 = 0.63, 즉 37% 할인)
  - 마트에서 100g당 단가 계산 시 비율 계산기를 활용하면 어떤 상품이 더 저렴한지 1초 만에 알 수 있습니다.
- **카드 2: 🎯 투자에서의 퍼센트 (아이콘: 📈)**
  - 주식이 -50% 떨어지면, 다시 원금을 회복하기 위해서는 +50%가 아닌 +100%가 올라야 합니다. 
  - 복리의 마법도 결국 퍼센트 누적의 결과입니다. 수익률 증감 모드를 통해 목표 금액을 쉽게 역산해 보세요.

### 1.10 자주 묻는 질문 (FAQ)
- **Q. 퍼센트 포인트(%p)와 퍼센트(%)의 차이는 무엇인가요?**
  - A. 기준 금리가 2.0%에서 2.5%로 올랐을 때, 이는 "0.5%p 올랐다"고 표현하는 것이 맞습니다. 만약 "퍼센트(%)"로 표현한다면 2.0에서 2.5로 변한 것이므로 "25% 증가했다"가 됩니다.
- **Q. 소수점은 몇 째 자리까지 계산되나요?**
  - A. 사용자의 편의를 위해 결과값은 소수점 최대 2~4째 자리까지 자동으로 반올림되어 깔끔하게 표시됩니다.
- **Q. 마이너스(-) 값도 입력이 가능한가요?**
  - A. 네, 손실율 계산이나 가격 하락 등을 계산할 때는 증감율 계산 모드에서 자연스럽게 파악이 가능하도록 지원합니다.

# property-기능명세서.md

## 1. 개요
Property Tax(재산세) 계산 메뉴의 기능 명세서입니다. 재산세는 일정한 재산에 대하여 부과되는 세금을 말합니다.

## 2. 화면 위치 및 파일 구조
1. **경로** : `/calculator/tax/property`
2. **파일 구성**
   - **서버**: `app/calculator/tax/property/page.tsx`
   - **클라이언트**: `app/calculator/tax/property/Property.tsx`

## 3. 계층 구조 및 SEO (Common Architecture)
1. **Breadcrumb Schema**: `홈 > 계산기 홈 > 세금 계산기 > 재산세 계산기` 구조의 JSON-LD 적용
2. **프리미엄 UI/UX 표준 (JIKO Grid)**:
   - **컨테이너**: `max-w-3xl mx-auto w-full px-4` 공간 제약 준수.
   - **카드 디자인**: 모든 입력창 및 결과 섹션은 `rounded-2xl`, `bg-white`, `shadow-xl` 적용.
   - **간격 표준**: 섹션 및 카드 간 간격은 **`mt-4`**로 통일하여 조밀하고 짜임새 있는 레이아웃 유지.
   - **결과 스크롤**: `useCalculatorScroll` Hook을 통한 결과 영역(`id="result-section"`) 자동 포커싱.

## 4. JIKO만의 차별화 기능 (Differentiation)
1. **공동명의 세액 분할**: 공동명의 선택 시 지분에 따른 인별 고지 금액 시뮬레이션.
2. **세부담 상한제 알림**: 직전 연도 세액 대비 상한(105~130%) 적용 가능성을 가이드라인으로 제시.
3. **도시지역분 통합 계산**: 재산세뿐만 아니라 잊기 쉬운 도시지역분 세액(0.14%)을 자동으로 합산.
4. **1세대 1주택 특례 적용**: 공정시장가액비율 인하(45% 이하) 및 특례 세율(0.05%p 인하) 자동 반영.
## 5. 화면 기능
### 1. 입력 데이터 (Input Data)
1. 대상 : 주택, 농지, 상가/건물, 토지
2. 1세대 1주택자 여부 : Y/N
3. 공동명의지분 비율 : %
4. 도시지역분 여부 : Y/N
5. 주택공시가격 : 단위 버튼(C, 1백, 5백, 1천, 5천, 1억, 5억, 10억), 주택일 경우 필수 입력
6. 토지공시지가 : 단위 버튼(C, 1백, 5백, 1천, 5천, 1억, 5억, 10억), 토지일 경우 필수 입력
7. 건물신축가격 : 단위 버튼(C, 1백, 5백, 1천, 5천, 1억, 5억, 10억), 상가/건물일 경우 필수 입력
10. 지역자원시설세 : 없음/중과/3중과, 상가/건물일 경우 필수 입력
* 대상별 입력 필수 항목 활성화 및 입력값 체크, 입력 불필요 항목 비활성화
* 필요시 입력필드 기준 설명 텍스트 추가


### 2. 출력 데이터 (Output Data)
1. 시가표준액
2. 과세표준
3. 재산세
4. 지방교육세
5. 지역자원시설세
6. 도시지역분
7. 총 납부세액
8. 결과




### 3. 버튼 기능 및 UX (JIKO Standard)
- **초기화 및 계산하기** : 공통 컴포넌트인 **`CalculatorButtons`** 적용
- **결과 복사 & 공유하기** : **`CalculatorActions`** 배치
- **결과 자동 스크롤**: `useCalculatorScroll` Hook을 통한 결과 영역 자동 포커싱

## 6. 하단 콘텐츠 (SEO 가이드 섹션)
1. **재산세 과세 대상 및 기준일**: 매년 6월 1일 소유자 기준 부과 안내.
2. **재산세 납부 기간**: 1기분(7월: 건물/주택1/2), 2기분(9월: 토지/주택1/2).
3. **공정시장가액비율**: 주택(43~45%), 상가/토지(70%) 등 과세표준 산정 비율 설명.
4. **재산세 세율표**: 주택별 세율(0.1~0.4%) 및 1주택 특례 세율 정리.
6. **사용 방법 (Usage)**: 
   - 보유한 부동산 자산의 종류(주택, 토지 등)를 선택합니다.
   - 당해 연도 공시가격(시가표준액)을 입력합니다.
   - 1세대 1주택 특례 대상자 여부를 체크하여 낮은 세율 적용을 확인합니다.
7. **계산 예시 (Example)**: 
   - 공시가격 5억원인 아파트를 보유한 1주택자의 경우: 공정시장가액비율(45%)이 적용된 과세표준 2.25억원에 대해 약 20만원 초반대(지방교육세 등 포함)의 재산세가 산출됩니다.
8. **주의사항 (Caution)**: 
   - 재산세는 매년 6월 1일 소유자에게 부과됩니다. 6월 1일에 등기를 치르면 해당 연도 세금이 부과되므로 매수 시 주의가 필요합니다.
   - 주택분 재산세액이 20만원을 초과하면 7월과 9월에 절반씩 나누어 부과됩니다.
9. **FAQ**: 
   - 6월 1일 이전에 집을 팔면 누가 재산세를 내나요?
   - 고지서가 7월과 9월 두 번 오는데 잘못된 건가요?
   - 공동명의가 단독명의보다 유리한가요?

## 7. 핵심 계산 로직
1. **과세표준** = 시가표준액(공시가격) * 공정시장가액비율
2. **재산세(산출세액)** = (과세표준 * 세율) - 누진공제
3. **지방교육세** = 재산세 * 20%
4. **지역자원시설세** = (재산세 과세표준과 별개로 건물분 등에 부과)
5. **도시지역분** = 과세표준 * 0.14%
6. **총 납부액** = 재산세 + 교육세 + 시설세 + 도시지역분
* 1세대 1주택자는 공시가격 구간별로 0.05%p 낮은 특례 세율 적용.

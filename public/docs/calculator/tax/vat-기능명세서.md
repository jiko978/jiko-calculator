# vat-기능명세서.md

## 1. 개요
VAT(부가가치세) 계산 메뉴의 기능 명세서입니다. 부가세 계산기로 합계금액에서 공급가액과 부가세(10%)를 자동으로 계산하세요. 세금계산서 발행, 사업자 정산, 견적서 작성 시 필수! 역산(합계→공급가액)과 순산(공급가액→합계) 양방향 계산을 지원하며, 계산 결과를 공유할 수 있습니다.

## 2. 화면 위치 및 파일 구조
1. **경로** : `/calculator/tax/vat`
2. **파일 구성**
   - **서버**: `app/calculator/tax/vat/page.tsx`
   - **클라이언트**: `app/calculator/tax/vat/Vat.tsx`

2. **프리미엄 UI/UX 표준 (JIKO Grid)**:

## 4. JIKO만의 차별화 기능 (Differentiation)
1. **과세 유형별 맞춤 계산**: 일반과세자(10%)와 간이과세자(업종별 부가가치율 15~40%)를 선택하여 계산 가능.
2. **역산/순산 원터치 전환**: 공급가액 기준과 합계금액 기준 계산을 탭 형태로 제공하여 사용자 편의성 극대화.
3. **업종별 부가가치율 가이드**: 간이과세자를 위한 업종별(소매, 음식, 숙박 등) 최신 요율 툴팁 제공.
4. **결과 시각화**: 전체 금액 중 실제 부가세가 차지하는 비중을 도넛 차트나 프로그레스 바로 표시.

- **공급 금액(부가세 별도/포함 선택)**: 단위 버튼(C, 1만, 5만, 10만, 50만, 1백, 5백, 1천, 5천, 1억, 5억)


### 2. 출력 데이터 (Output Data)
### 2. 출력 데이터 (Output Data)
- **공급가액**: 세액을 제외한 실제 물품/서비스 가액

## 6. 하단 콘텐츠 (SEO 가이드 섹션)
1. **부가세 계산 안내**: 순산(공급가액 -> 합계금액), 역산(합계금액 -> 공급가액)
2. **사용 방법 (Usage)**: 
   - '일반과세자' 또는 '간이과세자' 중 본인의 과세 유형을 선택합니다.
   - '순산(공급가액 기준)' 또는 '역산(합계금액 기준)' 탭을 선택합니다.
   - 해당 금액을 입력하면 공급가액, 부가세, 합계금액이 실시간으로 계산됩니다.
3. **계산 예시 (Example)**: 
   - 1대당 1,100,000원(부가세 포함)인 제품의 공급가액을 알고 싶을 때: 역산 기능을 이용하면 공급가액 1,000,000원, 부가세 100,000원으로 자동 분리됩니다.
4. **주의사항 (Caution)**: 
   - 간이과세자는 업종마다 부가가치율이 다르므로 주의하여 선택해야 합니다.
   - 면세 대상 품목(미가공 식료품 등)은 부가세가 발생하지 않으므로 계산에서 제외해야 합니다.
5. **FAQ**: 
- 부가세(VAT)란?
  ㄴ 부가가치세(VAT, Value Added Tax)는 상품이나 서비스의 생산 및 유통 과정에서 발생하는 부가가치에 대해 부과되는 간접세입니다. 한국의 부가세율은 10%이며, 최종 소비자가 부담하고 사업자가 대리 납부합니다.
- 부가세는 왜 필요한가요?
  ㄴ 부가세는 국가 재정의 중요한 수입원이며, 사회 기반 시설 확충 및 공공 서비스 제공에 사용됩니다.
- 부가세 면제 항목이 있나요?
  ㄴ 네, 일부 품목과 서비스는 부가세가 면제됩니다. 대표적으로 가공되지 않은 농산물·축산물·수산물, 의료·교육 서비스, 금융·보험 서비스, 도서·신문·잡지 등이 부가세 면제 대상입니다.
- 부가세 역산/순산 계산이란?
  ㄴ 부가세 역산 계산이란 합계금액에서 부가세를 제외한 공급가액을 계산하는 것이고, 순산 계산이란 공급가액에 부가세를 더한 합계금액을 계산하는 것입니다.

## 7. 핵심 계산 로직
1. **일반과세자 순산 (공급가액 → 합계)**
   - 부가세 = 공급가액 * 0.1
   - 합계금액 = 공급가액 + 부가세
2. **일반과세자 역산 (합계 → 공급가액)**
   - 공급가액 = 합계금액 / 1.1
   - 부가세 = 합계금액 - 공급가액
3. **간이과세자 (납부세액 기준)**
   - 납부세액 = 공급대가(합계액) * 업종별 부가가치율 * 10%
   - *단, 2021년 개정법에 따라 간이과세자도 발행 주체에 따라 부가세 포함 금액으로 표기 권장.*

JIKO-calculator-화면설계서

각 계산기 메뉴별 공통 화면설계서이다.

1. 화면 구성 (Layout)
전체 레이아웃은 모바일 퍼스트(Mobile-First)를 지향하며, PC/태블릿 환경에서는 중앙 정렬된 카드/박스 형태(`max-w-3xl mx-auto`)로 제공됩니다. 전체 배경은 부드러운 회색/다크모드(`bg-gray-50 dark:bg-gray-900`)를 사용하고, 각 영역은 독립된 카드 형태(`bg-white dark:bg-gray-800 rounded-3xl shadow-xl`)로 구분하여 시각적 피로도를 낮춥니다.

1.1 뒤로가기 - (화면이름) - 공유 버튼 (공통)
- 위치: 화면 최상단 고정 (Sticky Header)
- 배경: 화면 상단 고정, 100vh, background: white, width: 100%
- 구성:
  - 뒤로가기 아이콘: 뒤로가기
  - 현재 계산기 이름(또는 로고)
  - 공유 아이콘: 각 계산기별 고유 공유 링크


1.2 카테고리 메뉴명 (공통)
- 이모지 + 메뉴명
- 타이틀: 해당 계산기의 성격을 나타내는 이모지와 텍스트 뱃지 (예: `📈 복리 계산기`)

1.3 탭 구분 (옵션)
- 유사 메뉴 이동탭(예, 부동산 계산기의 DTI, DSR, LTV 계산기 등)
- 메뉴내 유사 기능 이동탭(예, 금융 계산기의 복리 계산기 등)
- UI 특징: 모바일에서 가로 스크롤을 지원하며 스크롤바는 숨김(`hide-scrollbar`). 선택된 탭은 Primary Color(파란색) 강조 처리.



1.4 입력 데이터 영역 카드 세션 (공통)
- 배경: background: white, width: 100%
- 구성:
  - 각 메뉴별 입력 데이터 영역
  - 초기화, 계산하기 버튼



1.5 출력 데이터 영역 카드 세션 (공통)
- 배경: background: white, width: 100%
- 구성:
  - 각 메뉴별 출력 데이터 영역
  - 결과 그래프, 테이블 등(옵션)
  - 결과 복사하기, 친구에게 공유하기 버튼

1.6 계산기 메뉴 설명 카드 세션 (공통)
- 배경: background: white, width: 100%
- 구성:
  - 각 메뉴별 설명 영역

1.7 사용 방법/계산 예시 카드 세션 (공통)
- 배경: background: white, width: 100%
- 구성:
  - 2단 레이아웃
  - 왼쪽 영역: 각 메뉴별 사용 방법
  - 오른쪽 영역: 각 메뉴별 계산 예시





1.8 각 메뉴별 추가 카드 세션 (공통)
- 배경: background: white, width: 100%
- 구성:
  - 각 메뉴별 추가 카드 영역
  - 최소 1개 이상 카드 세션(각 메뉴별로 추가 카드 세션이 다를 수 있음)
  - 1개 일 경우 : 1단 레이아웃

1.10 자주 묻는 질문 (FAQ) 카드 세션 (공통)

  - 각 메뉴별 FAQ 영역
  - 최소 3개 이상 구성

1.11 다른 금융 계산기도 확인해보세요! 카드 세션 (공통)

2.  기능 구성
2.1 계산하기 버튼
- 클릭시 계산 로직 실행
- 유효성 체크 실행
- 성공시 출력 영역 카드에 결과 표시
- 실패시 유효성 체크 메시지 표시

2.2 초기화 버튼
- 클릭시 입력 데이터 영역 초기화

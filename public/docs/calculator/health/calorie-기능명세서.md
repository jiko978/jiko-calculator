# calorie-기능명세서.md

## 1. 개요
칼로리(Kcal) 계산 메뉴의 기능 명세서입니다.

## 2. 화면 위치
1. 경로 : `/calculator/health/calorie`
2. 파일
ㄴ 서버 : `app/calculator/health/calorie/page.tsx`
ㄴ 클라이언트 : `app/calculator/health/calorie/Calorie.tsx`

## 3. 계층 구조 및 SEO (Common Architecture)
1. **Breadcrumb Schema**: `홈 > 계산기 홈 > 건강 계산기 > 칼로리 계산기` 구조의 JSON-LD 적용
2. **메타데이터**: '칼로리 계산기 · 운동 칼로리 소모 계산기 · 다이어트 식단 | JIKO 계산기' Title 적용
3. **PWA 연동**: 
   - `PWAInstallProvider`를 통한 설치 가능 상태 감지
   - `InstallBanner` (페이지 하단), `Footer` (앱 설치 버튼) 연동
4. 뒤로가기 버튼 / 공유 버튼 (NavBar 컴포넌트 필수 사용)
5. JIKO 플랫폼 홈(`jiko.kr`) 허브 이동 버튼 (Header 및 Footer 영역에 단축 위치)

## 4. 화면 기능
1. 입력 데이터
ㄴ 체중(kg) - 필수
ㄴ 운동 시간(분) 및 강도(METs) 직접 입력 또는 리스트 선택
ㄴ 주요 운동 종류 (METs 리스트):
   - 걷기, 달리기, 수영, 자전거, 줄넘기, 등산, 배드민턴, 볼링, 탁구 등 생활 체육 포함
ㄴ 주요 음식 종류 (kcal/serving 리스트):
   - 쌀밥, 단백질류, 과일류 뿐만 아니라 치킨, 떡볶이 등 일상 음식 포함
ㄴ 음식별 섭취량 선택 (1인분, 0.5인분 등)

2. 출력 데이터
ㄴ 소모 칼로리(Burned) 합계
ㄴ 섭취 칼로리(Intake) 합계
ㄴ **에너지 밸런스 시각화**: 섭취 vs 소모 비교 바 차트 및 순 증감분 안내
ㄴ 결과 복사 기능 (텍스트 클립보드 복사)
ㄴ 올바른 식생활 지침 및 영양 성분 팁 (WHO 가이드 적용)
ㄴ 결과 복사 기능 (텍스트 클립보드 복사)

3. 버튼 기능
ㄴ 초기화 : 모든 입력 데이터 삭제
ㄴ 계산하기 : 계산 결과 출력
ㄴ 결과 복사 : 결과 결과 복사

4. UI/UX 기능
ㄴ 초기화 버튼 클릭 시 흔들림 애니메이션 (shake) 효과 적용
ㄴ **표준 검증 UI**: 체중 미입력 등 필수값 누락 시 에러 핸들링 및 시각적 피드백 제공

5. 하단 정보 영역 (SEO 및 사용자 가이드 - page.tsx)
ㄴ 검색 엔진 최적화(SEO)를 위한 JSON-LD 라우팅 보강
ㄴㄴ `SoftwareApplication` 및 `FAQPage` 외 추가 범용 `schema` 스크립트 작성 및 로드
ㄴㄴ 구글 서치 콘솔(GSC) 등록 완료 (루트 경로 HTML 소유권 확인)
ㄴㄴ `sitemap.ts` 및 `robots.ts`를 통한 동적 크롤링 및 인덱싱 허용 (검색 노출 최적화)
ㄴ UI 디자인 : 본체와 동일한 카드형 레이아웃 및 다크모드 대응 적용
ㄴ 구성요소
ㄴㄴ 소개(H1) : 칼로리 계산기 및 기초 운동/다이어트 밸런스 측정 (운동 소모, 음식 섭취, 다이어트 키워드 포함)
ㄴㄴ 사용 방법 : 체중, 운동 종류별 소모량, 음식 섭취량을 통한 다이어트 식단 가이드
ㄴㄴ 계산 예시 : 1시간 달리기 시 소모 칼로리 및 식단 비교 예제
ㄴㄴ FAQ : 운동 소모 칼로리 계산 공식(METs), 다이어트 식단 팁, 칼로리 계산기와 기초대사량의 차이점 등 유사 키워드 포함 기획 (**공통 `FAQ` 컴포넌트 적용**)
ㄴㄴ 건강 계산기 더 보기 : 다른 건강 계산기 메뉴 바로가기 (**공통 `HealthMoreCalculators` 컴포넌트 적용**)

## 5. 계산 로직
1. 소모칼로리(kcal) = METs × 체중(kg) × (운동시간(분) / 60)
2. 섭취칼로리(kcal) = 음식별 칼로리 데이터 합계
3. 에너지 밸런스 지표 제공 (잉여/부족 칼로리)
4. 올바른 식생활 지침:
   - 나트륨 섭취 줄이기, 채소/과일 비중 높이기, 규칙적인 식사, 적정한 당분 섭취 등 (WHO 권장 사항)

## 6. 예외 처리
1. 입력값 검증 (정규식 필터링)
ㄴ 체중, 운동시간, 운동종류, 섭취음식, 음식종류는 숫자만 입력 가능 (문자 등 입력 시 무시)
ㄴ 체중, 운동시간, 운동종류, 섭취음식, 음식종류는 음수 입력 불가능
2. 계산 결과 검증
ㄴ 소모칼로리(kcal), 섭취칼로리(kcal)는 숫자만 출력

## 4. 화면 기능
1. 입력 데이터
ㄴ 나이
ㄴ 키(cm)
ㄴ 체중(kg)
ㄴ (선택) 활동 지수 (전혀 활동 없음 ~ 매우 활동적)
ㄴ (선택) 체지방률 (%) - 보다 정확한 Katch-McArdle 공식을 위해 선택 입력 가능

2. 출력 데이터
ㄴ 기초대사량(BMR) - Mifflin-St Jeor 공식 기준
ㄴ **BMR 비교 차트**: 연령대별 평균 대비 나의 위치 시각화 (0.7 ~ 1.3배 범위)
ㄴ **평균 데이터 테이블**: 연령대별 남/여 평균 BMR 정보 및 내 연령대 하이라이트
ㄴ 결과 복사 기능 (텍스트 클립보드 복사)

ㄴ 초기화 : 모든 입력 데이터 삭제
ㄴ 계산하기 : 계산 결과 출력
ㄴ 결과 복사 : 결과 결과 복사

4. UI/UX 기능
ㄴ 초기화 버튼 클릭 시 흔들림 애니메이션 (shake) 효과 적용

## 5. 계산 로직
1. 기초대사량(BMR) - Mifflin-St Jeor 공식 (표준):
   - 남성: (10 × 체중kg) + (6.25 × 키cm) - (5 × 나이) + 5
   - 여성: (10 × 체중kg) + (6.25 × 키cm) - (5 × 나이) - 161
2. 활동대사량(TDEE) = BMR × 활동 지수
   - 주로 앉아서 일함: 1.2
   - 가벼운 활동(주 1~3일): 1.375
   - 보통 활동(주 3~5일): 1.55
   - 활동적(주 6~7일): 1.725
   - 매우 활동적: 1.9

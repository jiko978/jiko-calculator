# dsr-기능명세서.md

## 1. 개요
DSR(총부채원리금상환비율) 계산 메뉴의 기능 명세서입니다. 복잡한 스트레스 DSR 단계별 적용과 대출 종류별 상환 원금을 정밀하게 계산하고, 현재 나의 부채 상태를 직관적으로 분석하며 최적의 상환 플랜을 설계하도록 돕습니다.

## 2. 화면 위치 및 파일 구조
1. **경로** : `/calculator/real-estate/dsr`
2. **동적 경로 (SEO)** : `/calculator/real-estate/dsr/[한글슬러그]` (예: /주택담보대출DSR, /직장인DSR한도)
3. **파일 구성**
   - **서버**: `app/calculator/real-estate/dsr/page.tsx`
   - **동적 서버**: `app/calculator/real-estate/dsr/[slug]/page.tsx`
   - **클라이언트**: `app/calculator/real-estate/dsr/Dsr.tsx`
   - **데이터**: `app/calculator/real-estate/data/products.json` (한글 슬러그 및 상품 정보 체계적 관리)

## 3. 계층 구조 및 SEO (Common Architecture)
1. **Breadcrumb Schema**: `홈 > 계산기 홈 > 부동산 계산기 > DSR 계산기` 구조의 JSON-LD 적용
2. **동적 메타데이터**: `[slug]`(대출 상품명)에 따라 Title, Description, Keywords 자동 생성
3. **한글 슬러그 처리**: URL 파라미터를 `decodeURIComponent` 및 `normalize('NFC')` 처리하여 검색엔진 유입 최적화.
4. **프리미엄 UI/UX 표준 (JIKO Grid)**:
   - **컨테이너**: `max-w-3xl mx-auto w-full px-4` 공간 제약 준수.
   - **카드 디자인**: 모든 입력창 및 결과 섹션은 `rounded-2xl`, `bg-white`, `shadow-xl` 적용.
   - **간격 표준**: 섹션 및 카드 간 간격은 **`mt-4`**로 통일하여 조밀하고 짜임새 있는 레이아웃 유지.
   - **결과 스크롤**: `useCalculatorScroll` Hook을 통한 결과 영역(`id="result-section"`) 자동 포커싱.

## 4. JIKO만의 차별화 기능 (Differentiation)
1. **DSR 위험도 게이지 (Visual Gauge)**
   - 계산 결과에 따라 **[안전(0~30%)] / [주의(30~40%)] / [위험(40% 초과)]** 상태를 시각적 게이지 바와 이모지로 표현.
2. **추가 대출 한도 시뮬레이터 (Reverse Math)**
   - DSR 40% 규제 한도 내에서 사용자가 현재 소득으로 **추가로 빌릴 수 있는 최대 대출 원금**을 자동 역산하여 제안.
3. **스트레스 DSR 단계별 비교 (Stress Phase Viewer)**
   - 1단계(시행 전), 2단계(현재), 3단계(완전 시행) 적용 시의 월 상환액 변화를 탭이나 비교 테이블로 즉시 확인.
4. **대출 종류별 원금 산정 자동화**
   - 신용대출(5년), 주담대(실제기간), 기타대출 등 종류별로 달라지는 DSR 부채 산정 만기 기준을 로직에 자동 반영.

## 5. 화면 기능
### 1. 입력 데이터 (Input Data)
- **대출구분**: 주택담보대출, 신용대출, 전세대출, 자동차담보대출, 예적금담보대출, 보험계약대출, 기타대출, 중도금 및 이주비, 전세보증금담보대출, 비주택부동산담보대출, 기타담보대출, 유가증권담보대출, 장기카드대출
- **연소득(원)**: (단위 버튼: c, 100만, 300만, 500만, 1000만, 3000만, 5000만 - 전용 규격 통일)
- **대출 원금(원)**: (단위 버튼 동일 제공)
- **대출 기간(개월/년)**: **6, 12, 24, 36개월 단축 선택 버튼** 제공
- **대출 금리(%)**: 소수점 2자리까지 입력 가능
- **대출 상환 방식**: 원리금균등, 원금균등, 만기일시상환 선택
- **보유 대출 연이자 상환액(원)**: (단위 버튼 동일 제공)
- **보유 대출 연원금 상환액(원)**: (단위 버튼 동일 제공)
- **스트레스 DSR 3단계 적용**: 변동형, 혼합형, 주기형 선택 및 단계별 가산금리 자동 연동

### 2. 출력 데이터 (Output Data)
- **신규 대출 월/연 원리금 상환액**: 대출 원금 + 총 이자 합계 (기간/금리/상환방식 기준)
- **보유 대출 월/연 원리금 상환액**: 기존 대출의 원금 + 이자 합계 정보
- **연소득 대비 DSR 스코어**: 연소득 / (신규 연 원리금 + 보유 연 원리금) * 100 및 위험도 배지 노출
- **스트레스 DSR 분석**: 수도권/비수도권 구분 및 3단계 적용 방식에 따른 연원리금 상환액 및 최종 DSR 비교 데이터
- **최대 추가 한도**: DSR 40% 도달 전까지 추가로 실행 가능한 최대 대출 원금 산출 결과

### 3. 버튼 기능 및 UX (JIKO Standard)
- **초기화 및 계산하기** : 공통 컴포넌트인 **`CalculatorButtons`** 적용 (전역 디자인 및 인터랙션 통일)
- **결과 복사 & 공유하기** : 코어 컴포넌트인 **`CalculatorActions`** 배치 (결과 텍스트 복사 및 카카오톡 공유)
- **결과 자동 스크롤**: 계산 완료 시 결과 영역(`id="result-section"`)으로 부드럽게 이동 (`useCalculatorScroll` Hook)
- **DSR 위험도 게이지**: 안전/주의/위험 단계별 시각적 게이지 바 노출

## 7. 핵심 계산 로직
- **원리금균등 상환액** = [대출원금 × 월금리 × (1 + 월금리)^기간] / [(1 + 월금리)^기간 - 1]
- **원금균등 상환액** = (대출원금 / 기간) + (남은 원금 × 월금리)
- **만기일시 상환액** = 월 이자만 납부 (대출원금 × 월금리)
- **DSR 계산** = 총 연간 원리금 상환액 / 연간 소득
- **스트레스 금리 가산** = 기준 금리 + (스트레스 금리 × 단계별 반영률[25%/50%/100%])

# ltv-기능명세서.md

## 1. 개요
LTV(담보인정비율) 계산 메뉴의 기능 명세서입니다. 복잡한 스트레스 DSR 단계별 적용과 대출 종류별 상환 원금을 정밀하게 계산하고, 현재 나의 부채 상태를 직관적으로 분석하며 최적의 상환 플랜을 설계하도록 돕습니다.

## 2. 화면 위치 및 파일 구조
1. **경로** : `/calculator/real-estate/ltv`
2. **동적 경로 (SEO)** : `/calculator/real-estate/ltv/[한글슬러그]` (예: /주택담보대출LTV, /직장인LTV한도)
3. **파일 구성**
   - **서버**: `app/calculator/real-estate/ltv/page.tsx`
   - **동적 서버**: `app/calculator/real-estate/ltv/[slug]/page.tsx`
   - **클라이언트**: `app/calculator/real-estate/ltv/Ltv.tsx`
   - **데이터**: `app/calculator/real-estate/data/products.json` (한글 슬러그 및 상품 정보 체계적 관리)

## 3. 계층 구조 및 SEO (Common Architecture)
1. **Breadcrumb Schema**: `홈 > 계산기 홈 > 부동산 계산기 > LTV 계산기` 구조의 JSON-LD 적용
2. **동적 메타데이터**: `[slug]`(대출 상품명)에 따라 Title, Description, Keywords 자동 생성
3. **한글 슬러그 처리**: URL 파라미터를 `decodeURIComponent` 및 `normalize('NFC')` 처리하여 검색엔진 유입 최적화.
4. **프리미엄 UI/UX 표준 (JIKO Grid)**:
   - **컨테이너**: `max-w-3xl mx-auto w-full px-4` 공간 제약 준수.
   - **카드 디자인**: 모든 입력창 및 결과 섹션은 `rounded-2xl`, `bg-white`, `shadow-xl` 적용.
   - **간격 표준**: 섹션 및 카드 간 간격은 **`mt-4`**로 통일하여 조밀하고 짜임새 있는 레이아웃 유지.
   - **결과 스크롤**: `useCalculatorScroll` Hook을 통한 결과 영역(`id="result-section"`) 자동 포커싱.

## 4. JIKO만의 차별화 기능 (Differentiation)
1. **규제 지역 자동 매핑 시스템**: 서울(강남3구/용산 등) 규제 지역 여부에 따른 LTV 상한선(40%~70%) 자동 추천.
2. **'방공제(소액임차보증금)' 자동 계산**: 지역별 최우선 변제 금액을 자동으로 한도에서 차감하여 '실수령 가능 한도' 제시.
3. **생애최초/영끌 보정 모드**: 생애최초 주택구입자 혜택(LTV 80%, 최대 6억 한도) 자동 적용 토글 제공.
4. **LTV-DSR 복합 진단 결과**: LTV 한도는 충족해도 DSR이 부족한 경우를 시각화하여 '빌릴 수 없는 돈'에 대한 경고 알람.

## 5. 화면 기능
### 1. 입력 데이터 (Input Data)
- **대출 목적**: 신규 주택 구입, 생활 안정 자금
- **지역 선택**: 규제 지역(강남3구, 용산), 비규제 지역(그외 수도권/광역시/기타)
- **보유 주택 수**: 
    - 구입 시: 생애최초, 무주택, 1주택, 2주택 이상
    - 생활안정: 1주택, 2주택 이상
- **주택 종류**: 아파트, 오피스텔, 빌라/다세대
- **주택 시세 (KB시세 기준)**: 원 단위 입력
- **희망 대출 금액**: 단위 버튼(C, 1백, 5백, 1천, 5천, 1억)
- **기존 부채 정보**: 연소득(DSR 연동용), 보유 주담대 잔액(생활안정자금 시)

### 2. 출력 데이터 (Output Data)
- **법정 LTV 한도(%)**: 규제 및 조건에 따른 최대 비율
- **산출 대출 한도(원)**: 시세 * 비율 - 방공제액(지역별 차등)
- **LTV 점유율 게이지**: 시각적 그래프 표현
- **DSR 예측 스코어**: 입력된 소득 기반 대출 가능 여부 1차 판정
- **실제 대출 가능액**: 규제 한도와 DSR 한도 중 최소값 가이드

### 3. 버튼 기능 및 UX (JIKO Standard)
- **초기화 및 계산하기** : 공통 컴포넌트인 **`CalculatorButtons`** 적용
- **결과 복사 & 공유하기** : **`CalculatorActions`** 배치
- **결과 자동 스크롤**: `useCalculatorScroll` Hook을 통한 결과 영역 자동 포커싱

## 6. 하단 콘텐츠 (SEO 가이드 섹션)
1. **소개 (Intro)**: LTV란?, 지역별 LTV 한도(40/50/70/80%), 방공제란 무엇인가?
2. **사용 방법 (Usage)**: 생애최초 혜택 조건 및 생활안정자금 한도 규정 가이드
3. **계산 예시 (Example)**: 규제 지역 내 10억 아파트, 1주택자 LTV 40% 적용 시 실제 한도 예시
4. **지역별 방공제 테이블**: 서울(5,500만), 수도권(4,800만), 광역시(2,800만) 등 최신 데이터 제공
5. **주의사항 (Caution)**: 
   - "DSR 기준 미달 시 LTV 한도만큼 대출이 승인되지 않을 수 있습니다." 명시
6. **FAQ**: 전세 퇴거 자금 대출 한도, 2주택자 처분 조건부 대출 등

## 7. 핵심 계산 로직
- **LTV 산출**: (주택 시세 * LTV 제한 비율) - (방공제액 * 대상 수)
- **LTV 제한 비율(규제 예시)**:
    - 비규제 무주택: 70%
    - 규제 지역 무주택: 50%
    - 생애최초: 80% (한도 6억)
- **실행 가능 한도**: Min(LTV 산출 한도, DSR 기반 역산 한도)

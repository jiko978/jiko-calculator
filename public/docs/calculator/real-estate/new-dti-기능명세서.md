# new-dti-기능명세서.md

## 1. 개요
new-DTI(신총부채원리금상환비율) 계산 메뉴의 기능 명세서입니다. 다주택자의 두 번째 주택담보대출 원리금까지 모두 부채로 합산하는 강화된 규제를 반영하여, 실질적인 대출 한도를 정밀하게 예측합니다.

## 2. 화면 위치 및 파일 구조
1. **경로** : `/calculator/real-estate/new-dti`
2. **동적 경로 (SEO)** : `/calculator/real-estate/new-dti/[한글슬러그]` (예: /주택담보대출신DTI, /신용대출신DTI)
3. **파일 구성**
   - **서버**: `app/calculator/real-estate/new-dti/page.tsx`
   - **동적 서버**: `app/calculator/real-estate/new-dti/[slug]/page.tsx`
   - **클라이언트**: `app/calculator/real-estate/new-dti/NewDti.tsx`
   - **데이터**: `app/calculator/real-estate/data/products.json`

## 3. 계층 구조 및 SEO (Common Architecture)
1. **Breadcrumb Schema**: `홈 > 계산기 홈 > 부동산 계산기 > 신DTI 계산기` 구조의 JSON-LD 적용
2. **동적 메타데이터**: `[slug]`(대출 상품명)에 따라 Title, Description, Keywords 자동 생성
3. **한글 슬러그 처리**: URL 파라미터를 `decodeURIComponent` 및 `normalize('NFC')` 처리하여 검색엔진 유입 최적화
4. **프리미엄 UI/UX 표준 (JIKO Grid)**:
   - **컨테이너**: `max-w-3xl mx-auto w-full px-4` 공간 제약 준수
   - **카드 디자인**: 입력창 및 결과 섹션은 `rounded-2xl`, `bg-white`, `shadow-xl` 적용
   - **간격 표준**: 섹션 간 간격은 **`mt-4`**로 통일하여 조밀한 레이아웃 유지
   - **결과 스크롤**: `useCalculatorScroll` Hook으로 결과 영역(`id="result-section"`) 자동 포커싱

## 4. JIKO만의 차별화 기능 (Differentiation)
1. **보수적 신DTI 게이지**
   - 계산 결과에 따라 **[안전/주의/규제위험]** 상태를 시각적 바와 안내 문구로 표현
2. **잠재 부채 반영 시나리오**
   - 카드론, 현금서비스, 미사용 한도, 변동금리 예측분 등을 포함한 **숨은 잠재부담**을 선택적으로 반영
3. **규제별 한도 비교**
   - **DTI vs 신DTI**를 동시에 보여주고, 신DTI 규제(예: 40% 또는 50% 규제 범위) 내 실제 여력 차이를 시각화
4. **시장 금융 규제 업데이트 반영**
   - 주담대/전세대출/전용보증 등 **상품별 신DTI 적용 대상을 자동 분류**하여 규제항목을 실시간으로 분리
5. **스트레스 금리 시뮬레이션**
   - 현재 금리와 **추가 가산금리 시나리오**를 비교하여 실제 승인 가능성에 더 근접한 보수적 한도 제시
1. **다주택자 한도 절벽 알람 (Multi-Mortgage Hub)**: 
   - 기존에 보유한 주담대 원금을 정확히 합산했을 때 한도가 얼마나 급격하게 줄어드는지 전/후 비교 리포트 제공.
3. **증빙소득 정밀 튜닝 (Income Multiplier)**:
   - 근로소득 외 소득 증빙 유무에 따른 가산율 또는 할인율을 반영하여 실제 은행 심사 통과 가능성 제시.
   

## 5. 화면 기능
### 1. 입력 데이터 (Input Data)
- **대출구분**: 주택담보대출, 신용대출, 전세대출, 자동차담보대출, 예적금담보대출, 보험계약대출, 기타대출, 카드론, 현금서비스
- **연소득(원)**: 단위 버튼 지원 (c, 100만~1억)
- **신규 대출 원금(원)**: 단위 버튼 및 자유 입력
- **대출 기간**: 6개월, 12개월, 24개월, 36개월, 또는 직접 입력
- **대출 금리(%)**: 소수점 2자리 입력 가능
- **상환 방식**: 원리금균등, 원금균등, 만기일시상환
- **보유 대출 연원리금 상환액(원)**: 기존 대출의 연간 원리금 합계
- **보유 대출 연이자 상환액(원)**: 기존 대출의 연간 이자 합계
- **잠재 부채 항목(선택)**: 카드론/현금서비스 미사용 한도, 예정된 추가 대출, 생활비 반영
- **신DTI 규제 유형**: 40% 규제, 50% 규제, 비규제 상품 구분
- **금리 스트레스 시나리오**: 현재 금리, +0.5%, +1% 가산 후 신DTI 변동 예측
- **대출 기간/금리/상환방식**: 공통 규격 준수
- **보유 주택담보대출 정보 (중요)**:
   - 기존 주담대 연간 원금 상환액(원)
   - 기존 주담대 연간 이자 상환액(원)
- **기타 대출 이자 상환액(원)**: 주담대 외 대출의 연간 이자 합계


### 2. 출력 데이터 (Output Data)
- **신DTI 비율**: 총 연간 원리금 상환액 + 잠재부채 예상액 / 연소득
- **DTI 대비 신DTI 차이**: 일반 DTI와 신DTI 결과 비교
- **규제 한도 도달 여부**: 30%/40%/50% 등 지역별 규제선 대조 결과
- **보수적 추가 한도**: 신DTI 40% 도달 전까지의 현실적인 추가 한도
- **시장 예측 리포트**: 주담대/전세대출/신용대출별 승인 가능성 요약
- **스트레스 시나리오 결과**: 가산금리 적용 후 신DTI 변화 그래프
- **보수적 신DTI 스코어**: 강화된 규제 기반 최종 수치 및 게이지
- **한도 축소분 분석**: 구DTI 대비 줄어든 대출 가능 금액 명시


### 3. 버튼 기능 및 UX (JIKO Standard)
- **초기화 및 계산하기**: 공통 컴포넌트 **`CalculatorButtons`** 적용
- **결과 복사 & 공유하기**: **`CalculatorActions`** 활용
- **결과 자동 스크롤**: 계산 완료 시 `id="result-section"`으로 이동
- **규제 경고 배지**: 신DTI 위험 구간에 진입하면 눈에 띄는 경고 배지 표시
- **시나리오 비교 탭**: 현재값 / 보수적값 / 스트레스값 탭으로 결과를 빠르게 전환

## 7. 핵심 계산 로직
- **원리금균등 상환액** = [대출원금 × 월금리 × (1 + 월금리)^기간] / [(1 + 월금리)^기간 - 1]
- **원금균등 상환액** = (대출원금 / 기간) + (남은 원금 × 월금리)
- **만기일시 상환액** = 월 이자만 납부 (대출원금 × 월금리)
- **신DTI 계산** = (총 연간 원리금 상환액 + 잠재 부채 예상액) / 연간 소득
- **DTI vs 신DTI 차이** = 신DTI - 일반 DTI
- **보수적 추가 한도** = (연소득 × 규제 한도) - 현재 신DTI 적용 연원리금
- **스트레스 금리 가산** = 현재 금리 + 스트레스 가산 금리

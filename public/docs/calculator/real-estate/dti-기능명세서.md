# dti-기능명세서.md

## 1. 개요
DTI(총부채원리금상환비율) 계산 메뉴의 기능 명세서입니다. 기존 부채와 신규 대출 원리금 부담을 모두 종합하여 **연소득 대비 DTI 비율을 정밀하게 계산**하고, 사용자에게 실제 대출 한도 감안 시나리오별 리스크와 추가 여력을 직관적으로 제시합니다.

## 2. 화면 위치 및 파일 구조
1. **경로** : `/calculator/real-estate/dti`
2. **동적 경로 (SEO)** : `/calculator/real-estate/dti/[한글슬러그]` (예: /주택담보대출DTI, /신용대출DTI)
3. **파일 구성**

2. **동적 메타데이터**: `[slug]`(대출 상품명)에 따라 Title, Description, Keywords 자동 생성

## 4. JIKO만의 차별화 기능 (Differentiation)
1. **DTI 위험도 게이지 (Visual Gauge)**
   - 계산 결과에 따라 **[안전(0~30%)] / [주의(30~40%)] / [위험(40% 초과)]** 상태를 시각적 바와 색상으로 표현
2. **한도 여력 역산 시뮬레이터 (Reverse Math)**:
   - 목표 DTI(예: 40%, 60%)까지 남은 여유 한도를 금액으로 자동 산출하여 제시.
3. **대출별 상환 부담 통합**
   - **주담대, 신용대출, 전세대출, 카드론, 기타대출**을 한 화면에서 통합 계산하여 DTI 산정 기준을 자동 반영
3. **상환 방식 체인지 뷰어**:
   - 원리금균등/원금균등/만기일시 선택에 따른 DTI 비율 변동을 실시간 차트로 시각화.
5. **생활비 차감형 시나리오**
   - 선택 항목으로 **기본 생활비(기본 공제)**를 반영한 예비 여력을 함께 표시하여 더 현실적인 DTI 결과 확인 가능
1. **DTI vs DSR 실시간 비교 (Gap Analyzer)**: 
   - DTI 결과와 함께 동일 소득 기준의 예상 DSR 수치를 병기하여 규제 차이를 직관적으로 파악.
4. **부채 구조 분석 (Debt Breakdown)**:
   - 신규 주담대 원리금 vs 기존 부채 이자의 비중을 고유 컬러로 구분하여 표시.


## 5. 화면 기능
### 1. 입력 데이터 (Input Data) - 단위 버튼(100만~5억) 필수 제공
- **대출구분**: 주택담보대출, 신용대출, 전세대출, 자동차담보대출, 예적금담보대출, 보험계약대출, 기타대출
- **연소득(원)**: 단위 버튼(c, 100만, 300만, 500만, 1000만, 5000만, 1억) 지원
- **신규 대출 원금(원)**: 단위 버튼 및 직접 입력 지원
- **대출 기간(개월/년)**: 6, 12, 24, 36개월 단축 버튼 및 직접 입력
- **대출 금리(%)**: 소수점 2자리 입력 가능
- **상환 방식**: 원리금균등 / 원금균등 / 만기일시상환 선택
- **보유 대출 연원리금 상환액(원)**: 기존 대출의 연간 원리금 상환액을 한 번에 입력
- **보유 대출 연이자 상환액(원)**: 기존 대출 연이자 항목을 별도로 입력
- **잠재 부채 입력(선택)**: 카드론/현금서비스 기대 월부담 또는 미사용 한도 반영
- **생활비 공제(선택)**: 기본 생활비를 차감한 DTI 적용 시나리오
- **신규 대출 종류별 적용 조건**: 주담대·전세대출 여부 및 대출 상품별 만기 기준 조정
- **기존 대출 연간 이자 합계(원)**: DTI의 핵심인 '이자만 합산' 규칙 적용



### 2. 출력 데이터 (Output Data)
- **DTI 비율**: 총 연간 원리금 상환액 / 연소득
- **안전 범위 및 리스크 상태**: 시각화된 위험도 텍스트와 바
- **예상 신규 대출 가능 금액**: 현재 DTI를 기준으로 허용 가능한 추가 대출 한도
- **기존 부채 통합 결과**: 기존 연 원리금 + 신규 연 원리금으로 산정된 종합 DTI
- **상환 방식별 월납 비교**: 원리금균등 vs 원금균등 vs 만기일시상환
- **생활비 차감 시나리오**: 기본 생활비를 반영한 실사용 가능 여력
- **상품별 규제 예측**: 주담대/전세대출/신용대출별 예상 DTI 영향
- **최종 DTI 스코어**: 위험도 배지 및 시각적 바 게이지 (0~100%)
- **위험도 등급**: [여유(0~40%)] / [관심(40~60%)] / [위험(60% 초과)]
- **신규 대출 연간 원리금**: 신규 신청 대출의 연간 부담액
- **추가 한도 제안**: 현재 DTI 기준 상한선까지 추가로 빌릴 수 있는 최대 원금 역산 결과
- **월 납입금 타임라인**: 상환 방식별 월평균 납입액 비교 리포트



### 3. 버튼 기능 및 UX (JIKO Standard)
- **초기화 및 계산하기**: 공통 컴포넌트 **`CalculatorButtons`** 적용
- **결과 복사 & 공유하기**: **`CalculatorActions`** 활용
- **결과 자동 스크롤**: 계산 완료 시 `id="result-section"`으로 이동
- **요약 카드**: 현재 DTI, 추가 여력, 권장 행동을 한눈에 보여주는 요약 카드 표시
- **단계별 팝업 도움말**: 각 입력 항목에 간단한 툴팁 문구 제공

## 7. 핵심 계산 로직
- **원리금균등 상환액** = [대출원금 × 월금리 × (1 + 월금리)^기간] / [(1 + 월금리)^기간 - 1]
- **원금균등 상환액** = (대출원금 / 기간) + (남은 원금 × 월금리)
- **만기일시 상환액** = 월 이자만 납부 (대출원금 × 월금리)
- **DTI 계산** = 총 연간 원리금 상환액 / 연간 소득
- **추가 한도 역산** = (연소득 × 목표 DTI) - 기존 연 원리금 상환액
- **생활비 차감 DTI** = 총 연 원리금 상환액 / (연소득 - 생활비)

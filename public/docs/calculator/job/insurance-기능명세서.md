# insurance-기능명세서.md

## 1. 개요
근로자와 사업주가 각각 부담하는 4대보험료를 정확히 산출하고, 각 보험 항목의 의미와 요율을 직관적으로 확인할 수 있도록 돕는 **4대보험 전용 계산기**의 기능 명세서입니다. 기존 '연봉/월급 계산기'가 '실수령액' 중심이었다면, 이 메뉴는 **'4대보험의 세부 내역과 사업주(회사) 부담금'**을 분석하는 데 중점을 두어 차별화합니다.

## 2. 화면 위치
1. 경로 : `/calculator/job/insurance`
2. 파일
   - 서버 : `app/calculator/job/insurance/page.tsx`
   - 클라이언트 : `app/calculator/job/insurance/Insurance.tsx`

## 3. 계층 구조 및 SEO (Common Architecture)
1. **Breadcrumb Schema**: `홈 > 계산기 홈 > 직장 계산기 > 4대보험 계산기` 구조의 JSON-LD 적용
2. **메타데이터**: '4대보험 계산기 | 2026 최신 근로자·사업주 보험료 계산 - JIKO 계산기' Title 적용
3. **PWA 연동**: 
   - `PWAInstallProvider`를 통한 설치 가능 상태 감지
   - `InstallBanner` (페이지 하단), `Footer` (앱 설치 버튼) 연동
4. 뒤로가기 버튼 / 공유 버튼 (NavBar 컴포넌트 필수 사용)
5. JIKO 플랫폼 홈(`jiko.kr`) 허브 이동 버튼 (Header 및 Footer 영역에 단축 위치)

## 4. 화면 기능
### 1. 입력 데이터 (4대보험 특화)
- **급여 기준 선택** : 연봉 / 월급
- **급여액 입력(원)** : 직접입력 (단위 버튼: 100만, 200만, 300만, 500만, 1000만, 3000만, 5000만, 1억 등 제공)
- **비과세액(식대 등)** : 기본 20만원 세팅, 직접입력 가능 (4대보험 산정 제외 금액)
- **[차별화] 기업 규모 선택** : 150인 미만 / 150인 이상 (고용보험 사업주 부담 요율 산정을 위해 필수)
- **[차별화] 산재보험료율** : 일반 요율(약 0.9%) 적용 (산재보험은 전액 사업주 부담임을 명시)
*(참고: 부양가족수 등 소득세 관련 항목은 UI 복잡도를 줄이기 위해 제거하거나 '소득세 간편 계산' 토글로 숨김 처리하여 4대보험 자체에 집중)*

### 2. 출력 데이터 (통계 및 시각화 지향)
- **월 납부액 요약 (Top Card)** :
  - **근로자 예상 부담금 합계** (월)
  - **사업주 예상 부담금 합계** (월)
- **[차별화] 회사 지원금(사업주 부담분) 통찰 카드** :
  - "회사가 나를 위해 추가로 납부하는 금액은 월 OOO원입니다!" 형태의 메시지로 진짜 연봉 가치 조명.
- **상세 내역 비교표 (근로자 vs 사업주)** :
  - 국민연금, 건강보험, 장기요양보험, 고용보험, 산재보험 각각에 대한 **요율 / 근로자 납부액 / 사업주 납부액 / 총 납부액** 테이블 제공.
- **[차별화] 도넛 차트 시각화** : 
  - 근로자 부담금 내 항목별 비율 차트 (국민연금 비중이 얼마나 큰지 등 시각적 제공)

### 3. 버튼 기능
- **초기화** : 모든 입력 데이터 삭제
- **계산하기** : 계산 결과 출력 및 하단으로 스크롤 이동
- **결과 복사** : 텍스트 형태로 나의 부담금과 사업주 부담금을 클립보드 복사

### 4. UI/UX 기능
- 초기화 버튼 클릭 시 흔들림 애니메이션 (`animate-shake`)
- **표준 검증 UI**: 필수 금액 누락 시 빨간색 테두리(`ring-red-500`) 및 흔들림 효과, 하단 에러 메시지 노출 (`animate-pulse`)
- 결과값이 나올 때 테이블과 차트가 부드럽게 나타나는 페이드인(`animate-fade-in-up`) 효과.

### 5. 하단 정보 영역 (SEO 및 사용자 가이드 - page.tsx)
- 검색 엔진 최적화(SEO)를 위한 JSON-LD 라우팅 보강
  - `WebApplication` 및 `FAQPage` 스키마 적용
- **[차별화] 4대보험 핵심 정보 가이드 (안내 카드)**
  - **건강/장기요양보험**: 요율 인상 및 혜택 안내
  - **국민연금**: 상한액/하한액 기준 안내 (월 소득이 상한액을 초과해도 일정 금액까지만 납부)
  - **고용/산재보험**: 실업급여 및 업무상 재해 보호 (산재는 100% 사업주 부담)
- 자주 묻는 질문 (FAQ) - **공통 `FAQ.tsx` 적용**
  1. *Q: 과세대상액(비과세 제외)이란 무엇인가요?*
  2. *Q: 산재보험은 왜 제 월급에서 공제되지 않나요?*
  3. *Q: 월급이 아무리 높아도 국민연금 공제액이 똑같습니다. 왜 그런가요? (상/하한액 설명)*
  4. *Q: 근로자 규모에 따라 4대보험료가 달라지나요? (고용보험 안내)*
- 직장 계산기 더 보기 (**공통 `JobMoreCalculators` 컴포넌트 적용**)

## 5. 계산 로직 (2025/2026 최신 기준)
1. **과세 대상액 산출**: `급여액 - 비과세액` (비과세액은 4대보험 산정 기준에서 제외됨)
2. **보험료 산정 (근로자 / 사업주 각각 계산)**:
   - **국민연금**: 과세대상액의 `4.5%` (근로자 4.5%, 사업주 4.5%). *단, 기준소득월액 상/하한액 컷오프 필수 반영.*
   - **건강보험**: 과세대상액의 `3.545%` (총 7.09%를 절반씩 부담). *상/하한액 컷오프 반영.*
   - **장기요양보험**: 산출된 건강보험료의 `12.95%` (근로자/사업주 동일 비율).
   - **고용보험**: 
     - 근로자: `0.9%` (실업급여 부담금)
     - 사업주: `0.9% (실업급여)` + `0.25% ~ 0.85% (고용안정·직업능력개발사업, 기업 규모에 따라 차등 적용, 기본 0.25% 세팅)`
   - **산재보험**: 과세대상액의 보험료율 (업종에 따라 다르나 평균 `1.5%` 내외, **전액 사업주 부담**).
3. 원단위 절사 (`Math.trunc(값 / 10) * 10`) 로직 적용하여 실제 공제 금액 단위를 맞춤.

## 6. 예외 처리
- 숫자가 아닌 문자 입력 방지 (정규표현식 `replace(/[^0-9]/g, '')`).
- 음수 입력 및 무리한 자리수 한도 초과(예: 100억 단위 이상) 방지.
- 비과세액이 총 급여를 초과할 경우, "비과세액은 급여를 넘을 수 없습니다" 경고 메시지.

## 7. 스타일(디자인) 참고

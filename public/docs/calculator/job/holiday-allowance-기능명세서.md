# holiday-allowance-기능명세서.md

## 1. 개요
주휴수당은 근로기준법에 따라 1주 동안 소정근로일을 개근한 근로자에게 지급되는 1일분의 임금입니다. 이 계산기는 **주휴수당의 정확한 금액을 산출**하고, **주휴수당 지급 대상 여부를 판별**하는 데 필요한 모든 요소를 체계적으로 제공합니다. 기존 '연봉/월급 계산기'가 '실수령액' 중심이었다면, 이 메뉴는 **'주휴수당의 세부 내역과 지급 대상 여부'**를 분석하는 데 중점을 두어 차별화합니다.

## 5. 계산 로직 (최신 근로기준법 기준)
1. **주휴수당 발생 조건 판별**
   - 사용자 입력 `주간근로시간`이 **15시간 미만**일 경우 → 주휴시간: 0, 주휴수당: 0원 (지급 대상 아님 알림 표시).
2. **주휴시간 계산**
   - 주 40시간 미만 근무 시: `(주간근로시간 / 40) × 8시간`
   - 주 40시간 이상 근무 시: 최대 **8시간**으로 고정 (초과 근무시간은 주휴시간에 반영되지 않음).
3. **금액 도출**
   - **주휴수당** = `주휴시간 × 입력 시급`
   - **주간 근로수당** = `주간근로시간 × 입력 시급`
   - **주급(총액)** = `주간 근로수당 + 주휴수당`
4. **월간 환산 산식 (1달 평균 4.345주 적용)**
   - 월간 근로시간 = `주간근로시간 × 4.345`
   - 월간 주휴시간 = `주휴시간 × 4.345`
   - **월간 주휴수당** = `주휴수당 × 4.345`
   - **월급(총액)** = `주급(총액) × 4.345` (소수점 첫째 자리에서 반올림 처리)

## 6. 예외 처리
- 숫자가 아닌 문자 입력 방지 (정규표현식 `replace(/[^0-9]/g, '')`).
- 음수 입력 및 무리한 자리수 한도 초과(예: 100억 단위 이상) 방지.
- 비과세액이 총 급여를 초과할 경우, "비과세액은 급여를 넘을 수 없습니다" 경고 메시지.

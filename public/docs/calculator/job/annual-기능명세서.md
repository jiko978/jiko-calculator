## 4. 화면 기능
1. **입력 데이터**
   - **(필수) 입사일 / 기준일(오늘)** : 달력(캘린더) UI 제공. 
   - **(선택) 사용 연차 일수(일)** : 일 단위 입력. default 0일.
   - **(선택) 월 통상임금(원)** : 기본급 + 고정 수당(식대, 직책수당 등) 입력. 통상임금 하단에 설명 추가.
   - **(선택) 1일 소정근로시간(시간)** : 시간 단위 입력. 기본 40시간 셋팅 및 수정 가능.

2. **출력 데이터**
   - **잔여 연차 시각화 (프로그레스 바)** : "총 *일 중 *일 사용, *일 남음"을 시각적인 UI로 표현.
   - **근속 기간** : 'X년 Y개월 Z일' 형태 표기
   - **총 발생 연차 누계** : 현재까지 발생한 모든 연차의 합.
   - **올해 발생 연차** : *년차 근무자 기준 올해 부여된 연차일수
   - **연차 발생 현황표 (History)** : 입사 시점부터 현재(또는 근속 5년 이내)까지의 연차 발생 내역을 타임라인 또는 표 형태로 제공. (1년 미만 월차 11개 포함)
   - **예상 미사용 연차수당(세전)** : 월 통상임금 입력 시 `(월 통상임금 ÷ 209시간 × 8시간) × 남은 연차일수` 공식을 통해 세전 예상액 출력.

3. **버튼 기능**
   - **초기화** : JIKO 코어 표준 규격(`handleReset`)을 사용하여 [화면 스크롤 상단 복귀] 및 폼 초기화.
   - **계산하기** : JIKO 코어 표준 규격(`useCalculatorScroll`)을 사용하여 유효성 검증 성공 시 결과 영역 스크롤 이동.
   - **결과 복사 & 공유하기** : 코어 컴포넌트인 `CalculatorActions`를 배치하여 2분할 버튼 형태로 통일 조치. (텍스트 복사 및 네이티브 공유 API 지원)
   - **상호 연동** : 연봉/월급 계산기, 실수령액 계산기 등으로 바로 이동 가능한 탭 제공.

4. **UI/UX 기능 (JIKO 코어 3대 표준 필수 적용)**
   - **표준 검증 UI**: 필수 항목(입사일 등) 누락 시 빨간색 테두리(`ring-2 ring-red-500/20`) 및 흔들림 효과(`animate-shake`), 버튼 상단 공통 에러 카드(`bg-red-50 animate-pulse`) 전시. 입력 수정(`onChange`) 시 즉각 상태 롤백.
   - **실시간 반응**: 날짜 선택 및 기준 옵션(입사일/회계년도) 변경 시 애니메이션을 동반하여 부가정보(직관적 배지 등) 즉각 제공.

5. **하단 정보 영역 (SEO 및 사용자 가이드 - page.tsx)**
   - **소개(H1)** : 2025년 기준 연차수당 계산기 및 지급 기준 안내.
   - **사용 방법** : 평균임금 산정 방식 및 상여금/연차수당 포함 기준 가이드.
   - **자주 묻는 질문(FAQ)** : 
     - 근로기준법 연차 발생 기준
     ㄴ 1년 미만 근로자 : 1개월 개근 시 1일의 유급휴가 발생, 최대 11일
     ㄴ 1년 이상 근로자 : 80% 이상 출근 시 15일의 유급휴가, 기본 15일
     ㄴ 3년 이상 근로자 : 2년마다 1일씩 가산, 최대 25일

     - 입사 1년차 연차 발생 기준
     ㄴ 입사 후 1년이 되기 전에 발생하는 연차를 흔히 '월차'라고 불러요. 
     ㄴ 1년 미만 근로자에게 주어지는 연차로, 1개월 개근 시 1일의 유급휴가가 발생하며 최대 11일까지 받을 수 있어요.
     ㄴ 1년차에는 11일의 월차가 발생하고, 1년이 지나면 15일의 연차가 발생해요.

     - 연차수당 계산 방법 (미사용 연차수당)
     ㄴ 근로기준법상 연차수당은 1일 '통상임금'의 100%를 지급하는 것이 원칙이에요.
     ㄴ 1일 통상임금 ≒ 월 통상임금 ÷ 209시간(주 40시간 근로자 기준) × 8시간
     ㄴ 연차수당 = 1일 통상임금 × 미사용 연차 일수
     ㄴ 예시: 월 통상임금이 300만 원이고 5일이 남았다면? (3,000,000 ÷ 209 × 8) × 5일 = 약 574,162원 (세전)

     - 연차수당 지급 기한
     ㄴ 퇴사일로부터 14일 이내에 지급해야 해요. 
     - 연차 발생 기준
     ㄴ 입사 1년 미만: 매월 1일 발생
     ㄴ 1년 이상: 80% 이상 출근 시 15일 발생
     ㄴ 3년 이상: 2년마다 1일씩 가산, 최대 25일
     - (**공통 FAQ 컴포넌트**).
   - **직장 계산기 더 보기** : 다른 직장 계산기 메뉴 바로가기 (**공통 JobMoreCalculators 컴포넌트**).

## 5. 계산 로직 (2025년 기준)
1. **재직일수 산출** : 기준일 - 입사일 계산 후 년/월/일 단위 파싱.
2. **법정 연차 발생 알고리즘** : 
   - 1년 미만: 매월 만근 시 1일씩 누적 (최대 11일).
   - 1년 이상: 기본 15일 부여.
   - 3년 이상: 매 2년마다 1일씩 가산 적용 (최고 한도 25일).
3. **회계연도 기준 변환 로직** : 1월 1일 기준으로 일할 계산하여 연차 부여 (입사일 기준보다 근로자에게 불리할 수 없음을 명시).

## 6. 예외 처리
1. **입력값 검증** : 숫자 필드 음수 입력 방지 및 날짜 교차 검증(`기준일 < 입사일` 인 경우 에러 카드 렌더링).
2. **통상임금 0원 처리** : 계산에 통상임금이 없다면 '연차수당' 영역은 렌더링하지 않거나 '정보 없음'으로 방어 처리.

## 7. 스타일(디자인) 참고

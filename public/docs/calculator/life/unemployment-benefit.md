## 4. 화면 기능
1. **입력 데이터**
   - **(필수) 생년월일 : 8자리 직접입력
   - **(필수) 장애인 여부 : 예/아니오(default 아니오)
   - **(필수) 재직기간 : '최종 퇴사일 이전 18개월 내 재직기간만 합산 가능' 문구 표기, 회사1 초기 셋팅(입사일, 퇴사일 달력 및 직접입력), 회사 삭제 버튼 회사1 우측 끝에 위치, 회사 추가 버튼 하단에 위치
   - **(필수) 최종월급(세전) : 직접입력, 퇴사전 3개월 동일 체크 버튼 추가(default 체크됨), 체크 버튼 해제시 1~3개월 전 월급 직접 입력 필드 생성

2. **출력 데이터**
   - **실업급여 수급 대상 여부** : 대상 / 대상 아님
   - **실업급여 수급 가능 일수** : *일
   - **실업급여 1일 예상 수령액** : *원
   - **실업급여 총 예상 수령액** : *원
   - **퇴직 당시 만 나이** : *세
   - 안내 문구
   ㄴ 실업급여는 퇴직 사유가 비자발적인 경우에만 지급됩니다.
   ㄴ 퇴사 당시 나이와 고용보험 가입 기간에 따라 실업급여를 받을 수 있는 기간이 달라집니다.
   ㄴ 퇴사 전 3개월간의 평균 월급을 기준으로 산정됩니다.
   ㄴ 수급 여부 및 정확한 금액은 반드시 관할 고용센터로 문의하시기 바랍니다.

3. **버튼 기능**
   - **초기화** : 모든 입력 데이터 삭제하며 흔들림(shake) 효과 적용.
   - **계산하기** : 필수 항목 검증 후 계산 결과 출력.
   - **결과 복사** : 
   - **실업급여 신청하기** : 고용보험 홈페이지 링크(https://m.work24.go.kr/cm/main.do)로 이동

4. **UI/UX 기능**
   - **표준 검증 UI**: 필수 항목(날짜, 급여) 누락 시 빨간색 테두리(Ring) 및 흔들림 효과, 하단 에러 메시지 노출 (Pulse 애니메이션).
   - **실시간 반응**: 날짜 선택 시 즉시 재직 기간을 상단에 배지로 표시하여 직관성 강화.

5. **하단 정보 영역 (SEO 및 사용자 가이드 - page.tsx)**
   - **소개(H1)** : 2025년 기준 실업급여 계산기 및 지급 기준 안내.
   - **자발적 퇴사시 실업급여 받는 방법안내** : (https://www.jobkorea.co.kr/GoodJob/Tip/View?News_No=18183 참고)
   - **실업급여 받는 방법 : 순서대로 나열(1~7번)
   - **사용 방법** : 실업급여 수급 대상 여부, 실업급여 수급 가능 일수, 실업급여 1일 예상 수령액, 실업급여 총 예상 수령액, 퇴직 당시 만 나이 등 안내.
   - **자주 묻는 질문(FAQ)** : 실업급여 수급 조건/대상, 실업급여 지급 기한, 실업급여 수급 기간, 실업급여 수급액 등 안내 (**공통 FAQ 컴포넌트**).
   - **생활 계산기 더 보기** : 다른 생활 계산기 메뉴 바로가기 (**공통 LifeMoreCalculators 컴포넌트**).

## 5. 계산 로직 (2025년 기준)
1. **재직일수 산출** : 퇴사일 - 입사일 (근속기간 확인).
2. **평균임금 계산** : (직전 3개월 급여 총액 + 연간 상여금 총액 * 3/12 + 연차수당 총액 * 3/12) / 직전 3개월 총 일수.
3. **법정 퇴직금** : (1일 평균임금 * 30일) * (총 재직일수 / 365).
4. **퇴직소득세** : 2025년 근속연수별 공제 및 세율 적용 (추정치).

## 6. 예외 처리
1. **입력값 검증** : 숫자만 입력 가능하도록 필터링, 음수 입력 방지.
2. **기간 검증** : 퇴사일이 입사일보다 빠른 경우 에러 메시지 노출.
3. **지급 대상 확인** : 재직일수가 1년(365일) 미만일 경우 "지급 대상이 아닙니다" 안내 문구 노출.

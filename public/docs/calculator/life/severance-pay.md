## 4. 화면 기능
1. **입력 데이터**
   - **(필수) 입사일 / 퇴사일** : 달력(캘린더) UI 제공. 입력 시 총 재직일수 및 퇴사 전 3개월 산정 기간 자동 표시.
   - **(필수) 월 급여 선택/입력** : 직전 3개월 월급 직접 입력 또는 평균값 자동 배분 기능 제공.
   - **(선택) 연간 상여금 총액** : 퇴사 전 1년간 지급받은 상여금 총액 (산정 시 3/12 자동 합산).
   - **(선택) 연차수당** : 퇴사 전 1년간 미사용으로 지급받은 연차수당 (산정 시 3/12 자동 합산).

2. **출력 데이터**
   - **총 재직일** : *년 *월 *일 (총 *일) 표기.
   - **평균임금 정보** : 1개월 평균임금 및 1일 평균임금 상세 노출.
   - **예상 퇴직금 (세전)** : 산정 공식에 따른 총 퇴직금액.
   - **예상 퇴직소득세** : 근속연수 공제 등이 반영된 추정 세액.
   - **실수령 퇴직금 (세후)** : 세금을 제외한 최종 수령 예상액.

3. **버튼 기능**
   - **초기화** : 모든 입력 데이터 삭제하며 흔들림(shake) 효과 적용.
   - **계산하기** : 필수 항목 검증 후 계산 결과 출력.
   - **결과 복사** : 세전/세후 금액 및 상세 공제 내역을 텍스트로 복사.
   - **상호 연동** : 연봉/월급 계산기, 실수령액 계산기 등으로 바로 이동 가능한 탭 제공.

4. **UI/UX 기능**
   - **표준 검증 UI**: 필수 항목(날짜, 급여) 누락 시 빨간색 테두리(Ring) 및 흔들림 효과, 하단 에러 메시지 노출 (Pulse 애니메이션).
   - **실시간 반응**: 날짜 선택 시 즉시 재직 기간을 상단에 배지로 표시하여 직관성 강화.

5. **하단 정보 영역 (SEO 및 사용자 가이드 - page.tsx)**
   - **소개(H1)** : 2025년 기준 퇴직금 계산기 및 지급 기준 안내.
   - **사용 방법** : 평균임금 산정 방식 및 상여금/연차수당 포함 기준 가이드.
   - **자주 묻는 질문(FAQ)** : 퇴직금 지급 기한(14일), IRP 계좌 이체 의무화, 중간정산 조건 등 (**공통 FAQ 컴포넌트**).
   - **생활 계산기 더 보기** : 다른 생활 계산기 메뉴 바로가기 (**공통 LifeMoreCalculators 컴포넌트**).

## 5. 계산 로직 (2025년 기준)
1. **재직일수 산출** : 퇴사일 - 입사일 (근속기간 확인).
2. **평균임금 계산** : (직전 3개월 급여 총액 + 연간 상여금 총액 * 3/12 + 연차수당 총액 * 3/12) / 직전 3개월 총 일수.
3. **법정 퇴직금** : (1일 평균임금 * 30일) * (총 재직일수 / 365).
4. **퇴직소득세** : 2025년 근속연수별 공제 및 세율 적용 (추정치).

## 6. 예외 처리
1. **입력값 검증** : 숫자만 입력 가능하도록 필터링, 음수 입력 방지.
2. **기간 검증** : 퇴사일이 입사일보다 빠른 경우 에러 메시지 노출.
3. **지급 대상 확인** : 재직일수가 1년(365일) 미만일 경우 "지급 대상이 아닙니다" 안내 문구 노출.

# 학점 계산기 (Grade Calculator) 기능 명세서

## 1. 개요
* **목적**: 대학생들의 현재 총 평점(GPA), 전공 평점 계산 및 목표 학점 달성을 위해 필요한 남은 학점을 계산해주는 서비스.

2. **파일**:

   - 클라이언트 : `app/calculator/life/grade/GradeCalculator.tsx`

3. **SEO 요소**: JSON-LD(SoftwareApplication), Semantic HTML5 적용.

## 4. 화면 기능 및 UX
### 4.1 과목 입력 폼 (동적 추가/삭제)
* **과목명**: 텍스트 입력 (선택 사항)
* **학점(Credit)**: 1~4학점 등 숫자 입력 (필수)
* **성적(Grade)**: A+, A0, B+, B0, C+, C0, D+, D0, F, P(Pass), NP(Non-Pass) 등 드롭다운 선택 (필수)
* **전공 여부**: 체크박스 (해당 과목이 전공인지 여부)
* **동적 제어**: '+ 과목 추가' 버튼을 통해 입력 행 추가, '-' 버튼을 통해 행 삭제. (기본 6개 행 제공)

### 4.2 출력 영역 (결과 리포트)
* **총 평점 & 전공 평점**: 직관적인 숫자로 강조 표시. 백분위 환산 점수 표기.
* **취득 학점 요약**: 이수 학점, 전공 학점, P/NP 학점 요약표 제공.
* **차트**: 만점 대비 현재 학점을 도넛 차트 형태로 시각화.
* **공통 UI 적용**: `resultRef`를 통한 Smooth Scroll 이동 (`useCalculatorScroll` 훅 사용). 결과 복사 및 공유 버튼 적용.

### 4.3 추가 기능
별도 탭으로 학점 변환기 제공
학점 변환기 입력 데이터
* 4.5 만점 / 4.3 만점 / 4.0 만점 / 100점 만점 선택 탭 버튼 (기본값: 4.5 만점)
* 변환할 점수 입력
* 변환 버튼 클릭
* 변환 결과 출력
ㄴ 각 만점별 변환 점수(총 4개) 출력
* *.** / 4.5만점
* *.** / 4.3만점
* *.** / 4.0만점
* *.** / 100점 만점
 
### 4.4 버튼 및 편의 기능
1. **초기화 및 계산하기** : 공통 컴포넌트인 **`CalculatorButtons`** 적용.
2. **결과 복사 & 공유**: `CalculatorActions` 연동.

### 4.5. 기준 평점(Scale) 설정
* 4.5 만점 / 4.3 만점 / 4.0 만점 / 100점 만점 선택 탭 버튼 (기본값: 4.5 만점)

### 4.6. 계산 로직 (Formulas)
* **이수 학점**: (P/NP 과목 포함) 수강한 모든 과목의 학점 총합
* **GPA 산출용 학점**: P/NP 과목을 제외한 학점 총합
* **총 평점(GPA)**: (각 과목 학점 × 성적 환산 점수)의 총합 / GPA 산출용 학점
* **전공 평점**: 전공 여부가 체크된 과목들의 (학점 × 성적) 총합 / 전공 GPA 산출용 학점


## 5. 하단 정보 콘텐츠 (SEO 가이드)
`page.tsx` 하단에 렌더링될 SEO 최적화 콘텐츠입니다.
* **H1 (가이드)**: 🎓 학점 계산기 가이드 (대학생 학점 관리 및 변환 안내)
* **H2 (사용 방법)**: 
  1. 본인 대학교의 기준 평점(4.5 / 4.3 / 4.0 / 100점 만점)을 선택합니다.
  2. 학점과 성적을 입력하고 전공 여부를 체크합니다.
  3. [학점 계산하기]를 누르면 총 평점과 전공 평점을 확인할 수 있습니다.
  4. 다른 만점 기준으로 변환이 필요할 경우 상단의 '학점 변환기' 탭을 이용하세요.
* **H2 (계산 예시)**: 4.5 만점 기준 전공 3학점(A+), 교양 2학점(B+) 입력 시의 계산 과정을 안내.
* **H2 (FAQ - 자주 묻는 질문)**:
  * Q. P/NP (Pass/Non-Pass) 과목은 어떻게 계산되나요?
    * A. 이수 학점에는 포함되지만, 총 평점(GPA) 산출을 위한 계산에는 제외됩니다.
  * Q. 전공 평점은 별도로 계산되나요?
    * A. 네, 전공으로 체크하신 과목들만 모아서 별도로 전공 평점을 계산해 드립니다.
  * Q. 4.5 만점과 4.3 만점의 차이는 무엇인가요?
    * A. 학교마다 A+에 부여하는 점수가 다르며, 4.3 만점의 경우 A- 등 세분화된 등급이 존재하는 경우가 많아 백분위 환산에 차이가 있습니다.

## 6. 예외 처리 및 유효성 검사 (Validation)
* 학점을 입력하지 않은 상태로 계산 시도 시 에러 표시 (빨간 테두리 + 흔들림 애니메이션 및 공통 에러 알림 카드 출력).
* 최소 1개 이상의 유효한 과목이 입력되어야 계산 처리.

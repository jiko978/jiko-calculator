# 날짜 계산기 기능 명세서 

## 1. 개요
두 날짜 사이의 정확한 기간을 계산하고, 특정 시점까지의 일수나 주수를 한눈에 확인하는 생활 밀착형 서비스입니다.

## 2. 화면 위치
1. **경로**: `/calculator/life/date`
2. **파일**:
   - 서버 : `app/calculator/life/date/page.tsx`
   - 클라이언트 : `app/calculator/life/date/Date.tsx`
3. **탭 내비게이션**: [날짜 계산기] | [디데이 계산기] | [전역일 계산기] (탭을 통해 상호 이동 가능)

## 3. 계층 구조 및 SEO

2. **메타데이터**: `날짜 계산기 | 날짜 일수 계산, 주수 계산, 개월수 확인 - JIKO 계산기`
3. **SEO 요소**: JSON-LD(SoftwareApplication), Semantic HTML5 적용.

## 4. 화면 기능 및 UX

### 4.1 입력 영역
1. **시작일**: 기본값 오늘 날짜. 달력(DatePicker) 선택 방식.
2. **종료일**: 기본값 오늘 날짜. 달력(DatePicker) 선택 방식.
3. **계산구분**: 사용자가 원하는 결과 형식을 버튼으로 선택
   - 일수(시작일 포함) : 전체 며칠인지 계산
   - 주수(시작일 포함) : 몇 주 몇 일인지 계산
   - 월수(시작일 포함) : 몇 개월 몇 일인지 계산
   - 년수(시작일 포함) : 몇 년 몇 개월 며칠인지 계산

### 4.2 출력 영역 (결과 리포트)
1. **핵심 결과**: 선택한 계산구분에 따른 기간 결과 표시 (예: "오늘부터 100일째 되는 날입니다")
2. **✨ 지코 차별화 포인트 (기념일 타임라인)**:
   - 계산 결과 하단에 해당 기간 내의 주요 마일스톤(100일, 200일, 300일, 1주년 등) 날짜를 리스트로 자동 자동 출력.
   - 예) "기다려지는 다음 기념일: 2026-05-20 (100일)"

### 4.3 버튼 및 편의 기능
1. **초기화 및 계산하기** : 공통 컴포넌트인 **`CalculatorButtons`** 적용.
2. **결과 복사 & 공유**: `CalculatorActions`를 통한 텍스트 복사 및 카카오톡 공유.
3. **자동 스크롤**: 계산 완료 시 결과 영역으로 부드러운 이동 (`useCalculatorScroll`).

## 5. 핵심 계산 로직
1. **초일산입 여부**: 기본적으로 '시작일 포함(초일산입)' 방식을 적용하되, 결과 하단에 기준을 명시함.
2. **윤년 처리**: 2월 29일이 포함된 연도의 경우 날짜 계산 라이브러리(date-fns 등)를 사용하여 오차 없이 계산.

## 6. 하단 정보 콘텐츠 (SEO 가이드)

### 6.1 날짜 계산 상식 TIP
- **초일불산입 원칙이란?**: 민법 제157조에 따르면 기간을 일, 주, 월 또는 연으로 정한 때에는 기간의 초일은 산입하지 않는 것이 원칙입니다. 하지만 연령 계산이나 특별한 규정이 있는 경우에는 초일을 산입합니다.
- **날짜 계산기 활용법**: 커플의 기념일, 아기의 생후 일수(개월수), 프로젝트 마감 기한 계산 등 다양한 생활 영역에서 활용할 수 있습니다.

### 6.2 FAQ
**Q. 만나이 계산과 날짜 계산 결과가 다른 이유는 무엇인가요?**
A. 나이 계산은 태어난 날부터 0세로 시작하여 매년 생일마다 나이가 올라가는 방식이며, 일반 날짜 계산은 순수하게 경과한 일수를 측정하기 때문입니다.

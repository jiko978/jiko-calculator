# 디데이 계산기 기능 명세서 

## 1. 개요
특정 기준일로부터 목표한 날까지 남은 기간을 계산하거나, 반대로 특정 일수 뒤 또는 전의 날짜를 계산해주는 생활 밀착형 서비스입니다.

## 2. 화면 위치
1. **경로**: `/calculator/life/d-day`
2. **파일**:
   - 서버 : `app/calculator/life/d-day/page.tsx`
   - 클라이언트 : `app/calculator/life/d-day/DDay.tsx`
3. **탭 내비게이션**: [날짜 계산기] | [디데이 계산기] | [전역일 계산기] (탭을 통해 상호 이동 가능)

## 3. 계층 구조 및 SEO

2. **메타데이터**: `디데이 계산기 | 기념일 계산, 목표일까지 남은 날짜, 목표일 계산 - JIKO 계산기`
3. **SEO 요소**: JSON-LD(SoftwareApplication), Semantic HTML5 적용.

## 4. 화면 기능 및 UX

### 4.1 입력 영역
1. **기준일**: 기본값 오늘 날짜. 달력(DatePicker) 선택 방식.
2. **목표일 설정 (A타입)**: 달력으로 특정 날짜 선택 시 남은 D-day 계산.
3. **일수 지정 (B타입)**: "기준일로부터 [ ]일 [전/후]는 언제인가요?"를 선택하여 직접 입력.

### 4.2 출력 영역 (결과 리포트)
1. **핵심 결과**: 
   - A타입: `D-100` 또는 `D+50` 형태의 대형 타이포그래피.
   - B타입: "2026-07-10 (금요일)"과 같은 타겟 날짜 정보.
2. **✨ 지코 차별화 포인트 (기념일 자동 리스트업)**:
   - 결과 카드 하단에 **기념일 한눈에 보기** 섹션 제공.
   - 기준일로부터 **100일, 200일, 300일, 1년, 2년, 500일, 1000일** 등 주요 날짜를 표 형식으로 미리 계산하여 출력.

### 4.3 버튼 및 편의 기능
1. **초기화 및 계산하기** : 공통 컴포넌트인 **`CalculatorButtons`** 적용.
2. **결과 복사 & 공유**: `CalculatorActions` 연동.
3. **자동 스크롤**: 계산 완료 시 결과 영역으로 부드러운 이동.

## 5. 핵심 계산 로직
1. **D-Day 계산 방식**: 목표일에 도달하면 'D-Day', 목표일 이후에는 'D+'로 표시하는 표준 로직 적용.
2. **날짜 계산**: 시간 단위를 00:00:00으로 고정하여 일 단위의 정확한 차이를 도출.

## 6. 하단 정보 콘텐츠 (SEO 가이드)

### 6.1 디데이 계산 활용 백서
- **커플 기념일**: 연애 시작일로부터 100일 또는 1주년이 언제인지 쉽고 빠르게 확인하세요.
- **수험생 D-Day**: 수능이나 중요한 자격증 시험일까지 남은 기간을 매일 체크하며 동기부여를 얻으세요.
- **아기 성장 기록**: 아기가 태어난 날부터 며칠째인지, 다음 예방접종일은 언제인지 확인하는 데 유용합니다.

### 6.2 FAQ
**Q. D-Day 계산 시 첫날을 포함하나요?**
A. 보통의 디데이 계산은 기준일을 0일로 보며(D-Day), 다음 날부터 1일로 칩니다. 하지만 상황에 따라 '그날부터 1일'로 치고 싶은 경우 날짜 계산기 탭을 활용해 일수 계산을 하시면 정확합니다.

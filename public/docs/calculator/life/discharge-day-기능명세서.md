# 전역일 계산기 기능 명세서 

## 1. 개요
군별 최신 복무 기간을 기준으로 입대일로부터 전역일을 정확히 산출하고, 복무율 및 주요 마일스톤(진급일, 전역일 카운트다운)을 시각적으로 확인하는 서비스입니다.

## 2. 화면 위치
1. **경로**: `/calculator/life/discharge-day`
2. **파일**:
   - 서버 : `app/calculator/life/discharge-day/page.tsx`
   - 클라이언트 : `app/calculator/life/discharge-day/DischargeDay.tsx`
3. **탭 내비게이션**: [날짜 계산기] | [디데이 계산기] | [전역일 계산기] (탭을 통해 상호 이동 가능)

## 3. 계층 구조 및 SEO
1. **Breadcrumb**: `홈 > 계산기 홈 > 생활 계산기 > 전역일 계산기`
2. **메타데이터**: `전역일 계산기 | 군대 전역일 계산, 복무율 확인, 진급일 계산 - JIKO 계산기`
3. **SEO 요소**: JSON-LD(SoftwareApplication), Semantic HTML5 적용.

## 4. 화면 기능 및 UX

### 4.1 입력 영역
1. **입대일**: 달력(DatePicker) 선택 방식.
2. **복무구분**: 군별 버튼군 (육군/해병, 해군, 공군, 사회복무요원, 의경, 해경, 소방 등).
3. **특이사항(옵션)**: 조기 전역 기간 또는 휴가를 미리 차감하고 싶은 사용자를 위한 일수 입력 필드.

### 4.2 출력 영역 (결과 리포트)
1. **핵심 결과**: "전역까지 D-200일", "전역일: 2027-12-31 (금요일)".
2. **✨ 지코 차별화 포인트 (복무율 & 진급 타임라인)**:
   - 전체 복무 기간 중 현재 어느 정도 왔는지 보여주는 **프로그레스 바 (복무율 %)**.
   - **이병 -> 일병 -> 상병 -> 병장**으로 넘어가는 자동 진급일 예측 리스트업.
   - 복무 단계별(초기, 중기, 말년) 맞춤 응원 메시지 노출.

### 4.3 버튼 및 편의 기능
1. **초기화 및 계산하기** : 공통 컴포넌트인 **`CalculatorButtons`** 적용.
2. **결과 복사 & 공유**: `CalculatorActions` 연동. (군인 친구나 애인에게 결과를 보내줄 때 유용)
3. **결과 자동 스크롤**: 계산 완료 시 결과 영역으로 부드러운 이동 (`useCalculatorScroll`).

## 5. 핵심 계산 로직
1. **군별 복무 기간**:
   - 육군/해병대: 18개월
   - 해군: 20개월
   - 공군: 21개월
   - 사회복무요원: 21개월
2. **날짜 계산**: 개월 단위 계산 시 날짜 라이브러리를 활용하여 정확한 전역일(입대일 시점 + N개월 - 1일) 산출.

### 6.2 FAQ
**Q. 휴가를 사용하면 전역일이 당겨지나요?**
A. 군법상 정식 전역일은 변하지 않지만, '전역 전 휴가'를 사용하여 실제 부대를 떠나는 날(사회로 복귀하는 날)을 계산할 때는 하단의 추가 옵션을 활용하여 남은 휴가 일수를 입력해 확인하실 수 있습니다.

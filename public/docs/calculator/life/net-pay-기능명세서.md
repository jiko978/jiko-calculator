실수령액(세후)을 기준으로 원하는 급여를 받기 위해 필요한 계약 연봉/월급(세전)을 추정하는 메뉴의 기능 명세서입니다.

## 4. 상세 기능 (Functionality)
1. **입력 항목**
   - **희망 실수령액 (월)** : 사용자가 실제로 통장에 찍히길 원하는 금액 (숫자 입력 및 단위 버튼)
   - **비과세액 (월)** : 식대 등 기본 20만원 세팅 (세금 계산에서 제외되는 항목)
   - **부양가족수 / 20세 이하 자녀수** : 공제액 산출을 위한 인적 공제 정보
   - **퇴직금 포함 여부** : 연봉 역산 시 1/12 또는 1/13 방식 선택

2. **출력 항목**
   - **추정 세전 월급** : 입력한 실수령액이 나오기 위해 필요한 월 기본급
   - **추정 계약 연봉** : 월급 기반으로 환산된 1년 총 급여 내역
   - **세금 및 공제 내역** : 역산 결과에 따른 국민연금, 건강보험, 고용보험, 소득세 상세 내역
   - **팁(Tip)** : "실수령액 350만원을 받으려면 연봉 약 5,000만원이 필요합니다" 식의 친절한 가이드 문구

3. **특이 사항**
   - **반복 역산 알고리즘(Iteration)** : 실수령액에서 세전을 한 번에 구하는 공식은 복잡하므로, 추정 범위를 좁혀가며 정확한 세전 금액을 찾는 방식 적용
   - **결과 공유** : "내가 원하는 실수령액을 위해 필요한 연봉" 정보를 카카오톡 등으로 공유 가능

## 5. 계산 로직 (2025년 기준)
1. **공제 요율** : 연봉/월급 계산기와 동일한 2025년 최신 요율 적용
2. **계산 방식** : 
   - 사용자가 입력한 `목표 실수령액`
   - 초기 `추정 세전금액` 설정 후 세금 계산 수행
   - 도출된 `계산된 실수령액`과 `목표 실수령액` 사이의 오차를 줄여가며 최종 세전 금액 도출 (오차범위 1원 미만까지)

## 6. UI/UX 디자인 (Interaction)
1. **연동 가이드 컴포넌트** : 
   - "혹시 세전 금액으로 실수령액이 궁금하신가요?" -> 연봉 계산기 바로가기 버튼 제공
2. **표준 검증 UI** :
   - 금액 미입력 시 빨간 테두리 및 흔들림 효과
   - 비과세액 초과 시 경고 메시지
3. **결과 시각화** :
   - 목표한 실수령액과 계산된 세전 금액의 비율을 그래프로 표시 (세금이 차지하는 비중)

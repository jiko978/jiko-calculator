# JIKO-calculator-프로그램명세서

이 문서는 JIKO 계산기(`app/calculator`) 프로젝트의 주요 소스 코드 리스트와 각 파일별 역할, 기술적 특성을 설명합니다.

## 1. 개요
- **기술 스택**: Next.js 15 (App Router), React, TypeScript, Tailwind CSS
- **핵심 특징**: 
  - 서버 컴포넌트 기반의 고성능 SEO 최적화
  - 하이브리드 슬러그(종목명-종목코드) 기반 동적 라우팅
  - PWA 지원 및 카카오톡 공유 최적화

## 6. 핵심 기술 로직

### 5.1 하이브리드 및 한글 슬러그 파싱
`[slug]/page.tsx`에서 사용되는 핵심 데이터 매핑 로직입니다.
1. **주식 하이브리드**: `삼성전자-005930` (이름과 코드를 분합)
2. **금융 한글 슬러그**: `decodeURIComponent`를 통해 URL 인코딩된 한글을 복원합니다.
3. **정규화 처리**: 브라우저/OS별 유니코드 차이 방지를 위해 `.normalize('NFC')`를 적용하여 데이터 매핑 신뢰성을 확보합니다.

### 5.2 정적 페이지 생성 (`generateStaticParams`)
한글 슬러그가 포함된 정적 페이지 생성을 위해 `encodeURIComponent`를 활용하여 안전한 파일 시스템 및 서버 경로를 확보합니다.

### 5.3 동적 뒤로가기 (`BackButton`)
`policy/` 레이아웃 등에서 사용자의 유입 경로(Referrer)를 분석하여 플랫폼 홈 또는 서비스 홈으로 지능적으로 이동하는 `BackButton` 컴포넌트를 제공합니다.

## 7. 제안 및 향후 확장 계획
- **상태 관리 최적화**: 현재 로컬 `state` 기반이나, 계산 이력 저장을 위해 `localStorage` 연동 제안.
- **이미지 자동 생성**: 공유 시 종목별 수익 현황을 캔버스로 그려 커스텀 공유 이미지 제공 기능 제안.
- **실시간 시세 연동**: 현재 사용자 입력 기반이나, 외부 API 연동을 통한 실시간 현재가 반영 제안.

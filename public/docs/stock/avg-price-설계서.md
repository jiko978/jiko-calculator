# 평균 단가 계산기 (avg-price) 설계서

## 1. 개요
주식 매수 시, 여러 차례에 걸쳐 다른 가격에 매수했을 때의 **최종 평균 단가 및 총 매수 금액**을 계산하는 메뉴입니다. (물타기/불타기 계산)

## 2. 화면 위치
- **경로**: `/stock/avg-price`
- **파일**: `app/stock/avg-price/page.tsx` (서버), `app/stock/avg-price/AvgPrice.tsx` (클라이언트)

## 3. 화면 UI/UX 구성
1. **입력폼 (리스트 형태)**
   - 차수별 입력 행 (1차 매수, 2차 매수, ...)
   - 입력 항목: 매수 단가 (원), 수량 (주)
   - '추가' 버튼: 새로운 차수의 매수 내역 입력 행 추가
   - '삭제' 아이콘: 해당 차수의 매수 내역 삭제 (단, 최소 1개의 행은 존재해야 함)
2. **제어 버튼**
   - **계산하기** : 입력된 데이터를 바탕으로 결과 도출
   - **초기화** : 모든 입력 행을 초기 상태(빈 1개 행)로 되돌림
3. **결과 영역**
   - 총 매수 수량 (주)
   - 총 매수 금액 (원)
   - **평균 매수 단가 (원)** (핵심 강조)

## 4. 상태(State) 관리
- `type Trade = { price: number | string; quantity: number | string }`
- `trades: Trade[]` : 사용자가 입력한 매수 내역 리스트 (초기값 길이 2개 정도 제공)
- `result: { totalQuantity: number; totalAmount: number; avgPrice: number } | null` : 계산 결과

## 5. 계산 로직
1. 매수 단가 × 수량 = 해당 차수 매수 금액
2. 총 매수 수량 = 각 차수 수량의 합
3. 총 매수 금액 = 각 차수 매수 금액의 합
4. **평균 단가 = 총 매수 금액 ÷ 총 매수 수량**

## 6. 예외 처리 가이드 (Validation)
- 매수 단가 또는 수량에 문자열/음수 입력 방지 (숫자만 허용, onChange에서 필터링)
- 빈 값이 포함된 상태에서 '계산하기' 클릭 시: 빈 값은 0으로 간주하거나, 계산에서 제외
- 총 수량이 0일 경우, 평균 단가를 Number.NaN 또는 무한대로 처리하지 않고 `0`으로 일괄 처리
